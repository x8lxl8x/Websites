
=== Правописание u - ó

<h1>Правописание u - ó</h1>
<p>Буквы <span class='clSrc'>u</span> и <span class='clSrc'>ó</span> обозначают один и тот же звук [u]. Их написание может быть объяснено особенностями исторического развития польского языка. Буква <span class='clSrc'>u</span> обозначает исконный звук [u], тогда как буква <span class='clSrc'>ó</span> обозначает звук, который произошел из гласного [o] (в других славянских языках, например, в русском, [o] сохранилось). На этой закономерности основываются конкретные правила написания <span class='clSrc'>u</span> - <span class='clSrc'>ó</span>:</p>
<p><span class='clSrc'>а)</span> <span class='clSrc'>ó</span> пишем в словах, если возможно чередование [u/o], например:</p>
<ul>
<li>sól - soli</li>
<li>stój - stoi</li>
<li>nie bój się - boi się</li>
<li>ból - boli</li>
<li>uspokój się - uspokoi się</li>
<li>nóg - noga </li>
<li>Bóg - Boga</li>
<li>ósmy - osiem</li>
<li>zawód - zawody</li>
<li>wytwór - wytwory</li>
<li>wiózł - wiozę</li>
<li>podróż - droga</li>
</ul>
<p><span class='clSrc'>б)</span> <span class='clSrc'>ó</span> пишем тогда, когда в похожих по звучанию словах в восточнославянских языках (русском, белорусском или украинском) этой букве соответствует <span class='clSrc'>о</span> или <span class='clSrc'>а</span>, хотя чередование [u/o] в польском языке не наблюдается, например:</p>
<ul>
<li><span class='clSrc'>chór</span> - хор</li>
<li><span class='clSrc'>miód</span> - мед</li>
<li><span class='clSrc'>góra</span> - гора</li>
<li><span class='clSrc'>kłócić</span> - колотить</li>
<li><span class='clSrc'>król</span> - король</li>
<li><span class='clSrc'>krótki</span> - короткий</li>
<li><span class='clSrc'>mózg</span> - мозг</li>
<li><span class='clSrc'>płótno</span> - полотно</li>
<li><span class='clSrc'>póki</span> - пока</li>
<li><span class='clSrc'>póty</span> - до тех пор</li>
<li><span class='clSrc'>późno</span> - поздно</li>
<li><span class='clSrc'>próba</span> - проба </li>
<li><span class='clSrc'>prócz</span> - кроме</li>
<li><span class='clSrc'>równy</span> - ровный</li>
<li><span class='clSrc'>różny</span> - разный</li>
<li><span class='clSrc'>stróż</span> - сторож</li>
<li><span class='clSrc'>tchórz</span> - хорек</li>
<li><span class='clSrc'>róża</span> - роза</li>
<li><span class='clSrc'>włóczyć</span> - тащить</li>
<li><span class='clSrc'>włókno</span> - волокно</li>
<li><span class='clSrc'>wróbel</span> - воробей</li>
<li><span class='clSrc'>wróżba</span> - ворожба</li>
<li><span class='clSrc'>źródło</span> - средство</li>
<li><span class='clSrc'>żółty</span> - желтый</li>
</ul>
<p>Исключение: <span class='clSrc'>ogórek</span> - огурец.</p>
<p><span class='clSrc'>в)</span> <span class='clSrc'>ó</span> пишем в окончаниях:</p>
<table>
<tr>
<td><span class='clSrc'>-ów</span> (в географических названиях)</td>
<td>Tarnów, Kraków, Chorzów, Janów</td>
</tr>
<tr>
<td><span class='clSrc'>-ów</span> (как окончание род.пад. мн.ч. существительных)</td>
<td>сhłopców, domów, bloków, projektów</td>
</tr>
<tr>
<td><span class='clSrc'>-ówna</span></td>
<td>Rodziewiczówna, Iłłakowiczówna, Bednarzówna</td>
</tr>
<tr>
<td><span class='clSrc'>-ówka</span></td>
<td>mieszkaniówka, placówka, żarówka, pocztówka</td>
</tr>
</table>
<p>Буква <span class='clSrc'>u</span> пишется:</p>
<p><span class='clSrc'>а)</span> в большинстве польских слов, которым в других славянских языках соответствуют слова со звуком [u], например:</p>
<ul>
<li><span class='clSrc'>artykuł</span> - статья, бел. «артыкул»</li>
<li><span class='clSrc'>szukać</span> - искать, бел. «шукаць»</li>
<li><span class='clSrc'>suchy</span> - сухой </li>
<li><span class='clSrc'>futro</span> - мех, бел. «футра»</li>
<li><span class='clSrc'>ucho</span> - ухо</li>
<li><span class='clSrc'>ulica</span> - улица</li>
</ul>
<p><span class='clSrc'>б)</span> в многочисленных суффиксах:</p>
<table>
<tr>
<td><span class='clSrc'>-un</span></td>
<td>zwiastun, opiekun</td>
</tr>
<tr>
<td><span class='clSrc'>-ulec</span></td>
<td>hamulec, budulec</td>
</tr>
<tr>
<td><span class='clSrc'>-uchna</span></td>
<td>matuchna, córuchna</td>
</tr>
<tr>
<td><span class='clSrc'>-utki</span></td>
<td>malutki, skromniutki</td>
</tr>
<tr>
<td><span class='clSrc'>-unek</span></td>
<td>wizerunek, stosunek</td>
</tr>
<tr>
<td><span class='clSrc'>-uch</span></td>
<td>leniuch, pastuch</td>
</tr>
<tr>
<td><span class='clSrc'>-uszek</span></td>
<td>staruszek, garnuszek</td>
</tr>
<tr>
<td><span class='clSrc'>-us</span></td>
<td>dzikus, lizus</td>
</tr>
</table>
<p><span class='clSrc'>в)</span> в формах глаголов, которые заканчиваются на <span class='clSrc'>-ować</span>, например: <span class='clSrc'>studiować</span> - <span class='clSrc'>studiuje</span>, <span class='clSrc'>rysować</span> - <span class='clSrc'>rysuje</span>, <span class='clSrc'>projektować</span> - <span class='clSrc'>projektuje</span>.</p>
<blockquote>
<p>Есть и некоторые отступления от этих правил. Запомните правильное написание польских слов, в которых на месте буквы <span class='clSrc'>u</span> в русском языке находится буква <span class='clSrc'>о</span> (в соответствии с правилом должно быть <span class='clSrc'>у</span>): </p>
<ul>
<li><span class='clSrc'>długi</span> - долгий</li>
<li><span class='clSrc'>zasuwać</span> - засовывать</li>
<li><span class='clSrc'>skuwać</span> - сковывать</li>
<li><span class='clSrc'>bruzda</span> - борозда</li>
<li><span class='clSrc'>dłuto</span> - долото</li>
<li><span class='clSrc'>kłuć</span> - колоть</li>
<li><span class='clSrc'>płukać</span> - полоскать</li>
<li><span class='clSrc'>pruć</span> - пороть</li>
<li><span class='clSrc'>but</span> - ботинок</li>
<li><span class='clSrc'>rum</span> - ром</li>
<li><span class='clSrc'>sum</span> - сом</li>
<li><span class='clSrc'>rumianek</span> - ромашка</li>
</ul>
</blockquote>
