
=== Способы выражения цели действия

<h1>Способы выражения цели действия</h1>
<ul>
<li><span class='clSrc'>dla</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>powiedzieć dla żartu</span> «сказать в шутку», <span class='clSrc'>zrobić dla matki</span> «сделать для матери»;</li>
<li><span class='clSrc'>do</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>ułożyć się do snu</span> «лечь спать», <span class='clSrc'>proszek do prania</span> «порошок для стирки белья», <span class='clSrc'>pasta do zębów</span> «зубная паста», <span class='clSrc'>służyć do pomocy</span> «служить для помощи», <span class='clSrc'>zmierzać do celu</span> «стремиться к мечте»;</li>
<li><span class='clSrc'>ku</span> + <span class='clSrc'>дат.пад.</span>, например: <span class='clSrc'>ku chwale ojczyzny</span> «во славу Родины»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>półka na książki</span> «полка для книг», <span class='clSrc'>lekarstwo na kaszel</span> «лекарство от кашля», <span class='clSrc'>melodia na flet</span> «мелодия для флейты», <span class='clSrc'>siatka na zakupy</span> «сетка для покупок», <span class='clSrc'>pójść na sekretarkę</span> «пойти работать секретаршей», <span class='clSrc'>nadawać się na żonę</span> «подходить / годиться в качестве жены», <span class='clSrc'>brać na świadka</span> «брать в качестве свидетеля», <span class='clSrc'>na rzecz</span> «на / в пользу», <span class='clSrc'>mieć na celu</span> «иметь своей целью».</li>
</ul>
<p>Конструкции с предлогом <span class='clSrc'>dla</span> обычно включают значение пользы (и могут толковаться как «в пользу кого-л. / чего-л.», например: <span class='clSrc'>1:0 dla Anderlechtu</span> - «1-0 в пользу Андерлехта». Они выражают также общее значение цели как мотива действия, тогда как конструкции с предлогом <span class='clSrc'>do</span> выражают конкретную цель - выполнение определенного действия, ср.: <span class='clSrc'>Dla ułatwienia pracy oddałem rękopis do przepisania na komputerze</span> - «Для облегчения работы я отдал рукопись для набора на компьютере»; <span class='clSrc'>Przysłano dokumenty do podpisania</span> - «Прислали документы на подпись»; <span class='clSrc'>Dla podpisania dokumentów musiałem przyjechać do biura</span> - «Для подписания документов мне нужно было пойти в контору». <span class='clSrc'>Dla</span> выражает обычно случайную, эпизодическую, а <span class='clSrc'>do</span> - постоянную цель, «объективное предназначение», например: <span class='clSrc'>nóż do konserw</span>, <span class='clSrc'>pokój do gry</span>.</p>
<p>Не существует и строгих правил, разграничивающих употребление предлогов <span class='clSrc'>do</span> и <span class='clSrc'>nа</span>, о чем свидетельствуют и параллельные сочетания: <span class='clSrc'>potrzebny do czegoś / na coś</span>, <span class='clSrc'>wystarczyć do czegoś / na coś</span>, <span class='clSrc'>skrzynka do listów / na listy</span>, <span class='clSrc'>kubeł do śmieci / na śmieci</span>. Первый предлог выражает предназначение предмета быть частью целого, например: <span class='clSrc'>kołnierz do palta</span>, <span class='clSrc'>guziki do bluzki</span>. При этом часто дополнительно передается значение «предмет создан / сделан для выполнения указанной функции», например: <span class='clSrc'>maszyna do pisania</span>. Предлог <span class='clSrc'>na</span> может дополнительно выражать значение «употребляется, в том числе и случайно, для выполнения указанного действия», например: <span class='clSrc'>puszka na niedopałki</span> - «банка для окурков». Предлог <span class='clSrc'>na</span> выражает также предназначение вещества быть материалом для изделия, например: <span class='clSrc'>skóra na buty</span>, <span class='clSrc'>mleko na zupę</span>.</p>
<ul>
<li><span class='clSrc'>o</span> + <span class='clSrc'>вин. пад.</span>, например: <span class='clSrc'>strajkować o cofanie podwyżek</span> «бастовать с требованием отменить повышение (цен)»;</li>
<li><span class='clSrc'>po</span> + <span class='clSrc'>вин. пад.</span>, например: <span class='clSrc'>po zakupy</span> «в магазин / за покупками», <span class='clSrc'>po książki</span> «за книгами», <span class='clSrc'>po pieniądze</span> «за деньгами», <span class='clSrc'>zwrócić się po radę</span> «обратиться за советом». Подобные конструкции встречаются в белорусском языке, но они не характерны для русского: бел. <em>пайшлі па ягады, палез па агуркі</em> - русск. «пошли за ягодами, полез за огурцами».</li>
</ul>
