
=== Способы выражения причины действия

<h1>Способы выражения причины действия</h1>
<ul>
<li><span class='clSrc'>o</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>kłócić się o talerzе</span> «ссорится из-за тарелки», <span class='clSrc'>byłem o to zły</span> «я злился из-за этого»;</li>
<li><span class='clSrc'>dla</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>porównują dla jego czarności do diabła</span> «из-за черноты его сравнивают с чертом», <span class='clSrc'>powody, dla których...</span> «причина, по которой…»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>umrzeć na tyfus</span> «умереть от тифа», <span class='clSrc'>na skutek awarii</span> «в результате аварии», <span class='clSrc'>na jego życzenie</span> «по его желанию», <span class='clSrc'>na prośbę robotników</span> «по просьбе сотрудников», <span class='clSrc'>na sygnał</span> «по сигналу», <span class='clSrc'>na samo wspomnienie</span> «при одном воспоминании», <span class='clSrc'>radość na myśl</span> «радость от мысли / при мысли»;</li>
<li><span class='clSrc'>na skutek</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>na skutek wzdrożenia</span> «в результате подорожания»;</li>
<li><span class='clSrc'>po</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>ślad po paznokciu</span> «след от ногтя», <span class='clSrc'>ślady po duszeniu</span> «следы после душения»;</li>
<li><span class='clSrc'>przez</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>spóżnić się przez dozorcę</span> «опоздать из-за сторожа», <span class='clSrc'>przez pomyłkę</span> «ошибочно», <span class='clSrc'>przez głupotę</span> «по глупости», <span class='clSrc'>przez upór</span> «из упрямства»;</li>
<li><span class='clSrc'>przed</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>chciał mówić, ale przed płaczem nie mógł</span> «он хотел говорить, но из-за слез не мог»;</li>
<li><span class='clSrc'>z/ze</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>tańczyć z radości</span> «танцевать от радости», <span class='clSrc'>drżeć z zimna</span> «дрожать от холода», <span class='clSrc'>krzyczeć z rozpaczy</span> «кричать от отчаяния», <span class='clSrc'>z konieczności</span> «по необходимости», <span class='clSrc'>z braku</span> «из-за недостатка», <span class='clSrc'>z powodu</span> «по причине», <span class='clSrc'>żałować z faktu malejącego zainteresowania</span> «сожалеть по той причине, заинтересованность уменьшается», <span class='clSrc'>słynny ze swoich pomyłek</span> «известный своими ошибками»;</li>
<li><span class='clSrc'>za</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>wyjść za naturalną potrzebą</span> «выйти по нужде», <span class='clSrc'>wrócił do kraju za ciągłym naleganiem ojca</span> «вернулся на родину после настоятельных просьб отца», <span class='clSrc'>za moim przybyciem twarz jej się zarumieniła</span> «после моего появления ее лицо покраснело»;</li>
<li><span class='clSrc'>z powodu</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>z powodu uroczystości</span> «по причине торжества», <span class='clSrc'>z byle powodu</span> «по любой причине».</li>
<li><span class='clSrc'>dzięki</span> + <span class='clSrc'>дат.пад.</span>, например: <span class='clSrc'>dzięki przypadkowi</span> «благодаря случаю», <span class='clSrc'>dzięki przyjaciołom</span> «благодаря друзьям», <span class='clSrc'>dzięki pomocy państwa</span> «благодаря поддержке государства»;</li>
<li><span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>przymierać głodem</span> «умирать с голода», <span class='clSrc'>zginęli dziwnym zbiegiem wypadków</span> «погиб при удивительном стечении обстоятельств», <span class='clSrc'>narody giną niesprawiedliwością</span> «народы гибнут от несправедливости».</li>
</ul>
