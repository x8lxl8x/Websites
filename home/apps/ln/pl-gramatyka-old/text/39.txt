
=== Словобразование наречий

<h1>Словобразование наречий</h1>
<p>Большинство наречий образовано от прилагательных. Этот класс слов можно разделить на простые и сложные формы:</p>
<p><span class='clSrc'>а)</span> простые образуются при помощи суффиксов:</p>
<p><span class='clSrc'>-o</span>: <span class='clSrc'>mały</span> - <em>mało</em>, <span class='clSrc'>duży</span> «большой» - <em>dużo</em>, <span class='clSrc'>tani</span> «дешевый» - <em>tanio</em>;</p>
<p><span class='clSrc'>-e</span>: <span class='clSrc'>dobry</span> «хороший» - <em>dobrze</em>, <span class='clSrc'>zły</span> «плохой» - <em>źle</em>, <span class='clSrc'>serdeczny</span> - <em>serdecznie</em>.</p>
<p>Твердых правил употребления этих суффиксов нет. Обычно при помощи суффикса <span class='clSrc'>-о</span> образуются формы наречий от прилагательных с основой на мягкий или затвердевший согласный (<span class='clSrc'>głupi</span> - <em>głupio</em>, <span class='clSrc'>świeży</span> - <em>świeżo</em>), а при помощи суффикса <span class='clSrc'>-e</span> - от прилагательных с твердой основой (<span class='clSrc'>złośliwy</span> «злобный» - <em>złośliwie</em>). О нерегулярности употребления суффиксов говорит тот факт, что от одного и того же прилагательного можно образовать наречия при помощи обоих суффиксов (<span class='clSrc'>wysoki</span> - <em>wysoko</em>, <span class='clSrc'>smutny</span> «грустный» - <em>smutno</em>, <span class='clSrc'>wysoki</span> - <em>wysoce</em>, <span class='clSrc'>smutny</span> - <em>smutnie</em>).</p>
<p><span class='clSrc'>б)</span> сложные наречия образуются при помощи:</p>
<ul>
<li>предлога <span class='clSrc'>po</span> и суффикса <span class='clSrc'>-u</span>: <span class='clSrc'>polski</span> - <span class='clSrc'>po</span> polsk<span class='clSrc'>u</span>, <span class='clSrc'>europejski</span> - <span class='clSrc'>po</span> europejsk<span class='clSrc'>u</span>. Следует обратить внимание на отсутствие дефиса после приставки;</li>
<li>предлога <span class='clSrc'>ро</span> и суффикса <span class='clSrc'>-emu</span>: <span class='clSrc'>stary</span> - <span class='clSrc'>po</span> star<span class='clSrc'>emu</span>, <span class='clSrc'>boży</span> - <span class='clSrc'>po</span> boż<span class='clSrc'>emu</span>;</li>
<li>предлога <span class='clSrc'>z</span> и суффикса <span class='clSrc'>-а</span>: <span class='clSrc'>polski</span> - <span class='clSrc'>z</span> polsk<span class='clSrc'>a</span>, <span class='clSrc'>daleki</span> - <span class='clSrc'>z</span> dalek<span class='clSrc'>a</span>;</li>
<li>предлога <span class='clSrc'>na</span> и суффикса <span class='clSrc'>-o</span>: <span class='clSrc'>prawy</span> - <span class='clSrc'>na</span> praw<span class='clSrc'>o</span>, <span class='clSrc'>gorący</span> «горячий» - <span class='clSrc'>na</span> gorąc<span class='clSrc'>o</span>;</li>
<li>предлога <span class='clSrc'>za</span> и суффикса <span class='clSrc'>-o</span>: <span class='clSrc'>zimny</span> «холодный» - <span class='clSrc'>za</span> zimn<span class='clSrc'>o</span>, <span class='clSrc'>gorący</span> «горячий» - <span class='clSrc'>za</span> gorąc<span class='clSrc'>o</span>; эти наречия связаны с обозначением чрезмерности в проявлении признака или свойства.</li>
</ul>
<p>От числительных наречия образуются при помощи предлога <span class='clSrc'>po</span> и суффикса <span class='clSrc'>-е</span>: <span class='clSrc'>pierwszy</span> - <span class='clSrc'>po pierwsze</span>, <span class='clSrc'>drugi</span> - <span class='clSrc'>po drugie</span>.</p>
<p>Наречия, образованные от существительных, делятся на простые и сложные:</p>
<p><span class='clSrc'>а)</span> простые - существительные в различных косвенных падежах без предлогов: <span class='clSrc'>razem</span> «вместе», <span class='clSrc'>czasem</span> «иногда», <span class='clSrc'>rankiem</span> «утром», <span class='clSrc'>kłusem</span> «рысью», <span class='clSrc'>gwałtem</span> «насильно»;</p>
<p><span class='clSrc'>б)</span> сложные - существительные в различных косвенных падежах с предлогами: <span class='clSrc'>wraz</span> «сразу же», <span class='clSrc'>raz po raz</span> «раз за разом», <span class='clSrc'>na czas</span> «временно», <span class='clSrc'>wówczas</span> «тогда», <span class='clSrc'>w przód</span> «раньше», <span class='clSrc'>naprzód</span> «вперед», <span class='clSrc'>powoli</span> «медленно», <span class='clSrc'>po trochę</span> «понемножку». Не существует твердых правил слитного и раздельного написания этих форм - можно встретить оба варианта (<span class='clSrc'>wprzód</span> и <span class='clSrc'>w przód</span>).</p>
