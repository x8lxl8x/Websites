
=== Словообразование глаголов

<h1>Словообразование глаголов</h1>
<p>В зависимости от того, от какой части речи образованы глаголы, все их можно разделить на следующие группы:</p>
<ol>
<li>Глаголы, образованные от существительных</li>
<li>Глаголы, образованные от прилагательных</li>
<li>Глаголы, образованные от глаголов</li>
</ol>
<h2>Глаголы, образованные от существительных</h2>
<p>Глаголы этой группы обозначают действие, которое так или иначе связано с производящим существительным:</p>
<ul>
<li><span class='clSrc'>-ić, -eć, -ać</span>: <span class='clSrc'>mowa</span> «речь» - <span class='clSrc'>mówić</span> «говорить», <span class='clSrc'>sąd</span> - <span class='clSrc'>sądzić</span>, <span class='clSrc'>szal</span> «бешенство» - <span class='clSrc'>szaleć</span> «беситься», <span class='clSrc'>ból</span> - <span class='clSrc'>boleć</span>, <span class='clSrc'>stuk</span> - <span class='clSrc'>stukać</span>, <span class='clSrc'>lęk</span> «страх» - <span class='clSrc'>lękać</span> «бояться»;</li>
<li><span class='clSrc'>-ować, -izować</span>: <span class='clSrc'>żart</span> «шутка» - <span class='clSrc'>żartować</span>, <span class='clSrc'>podróż</span> «путешествие» - <span class='clSrc'>podróżować</span>, <span class='clSrc'>melancholia</span> - <span class='clSrc'>melancholizować</span>;</li>
<li><span class='clSrc'>-nąć</span>: <span class='clSrc'>krzyk</span> - <span class='clSrc'>krzyknąć</span>, <span class='clSrc'>błysk</span> - <span class='clSrc'>błysnąć</span>, этот суффикс вносит значение однократности;</li>
</ul>
<h2>Глаголы, образованные от прилагательных</h2>
<p>Общее значение этих глаголов - значение действия, в результате которого возникает признак, названный производящим прилагательным:</p>
<ul>
<li><span class='clSrc'>-ić</span>: <span class='clSrc'>biały</span> - <span class='clSrc'>bielić</span> «делать белым», <span class='clSrc'>słodki</span> - <span class='clSrc'>słodzić</span>;</li>
<li><span class='clSrc'>-eć</span>: <span class='clSrc'>biały</span> - <span class='clSrc'>bieleć</span> «делаться белым», <span class='clSrc'>wesoły</span> - <span class='clSrc'>weseleć</span>;</li>
<li><span class='clSrc'>-nąć</span>: <span class='clSrc'>słaby</span> - <span class='clSrc'>słabnąć</span>, <span class='clSrc'>miękki</span> «мягкий» - <span class='clSrc'>mięknąć</span> «смягчаться»;</li>
<li><span class='clSrc'>-ować</span>: <span class='clSrc'>chory</span> - <span class='clSrc'>chorować</span>, <span class='clSrc'>próżny</span> «напрасный» - <span class='clSrc'>próżnować</span> «лениться»;</li>
</ul>
<h2>Глаголы, образованные от глаголов</h2>
<p>Чаще всего процесс словообразования в этом случае связан с изменением вида глагола, особенно это касается суффиксального способа образования. Однако видовую характеристику глагола изменяют также приставки, внося в значение смысловые оттенки:</p>
<ul>
<li><span class='clSrc'>w-</span>: <span class='clSrc'>płątać</span> «путать» - <span class='clSrc'>wpłątać</span> «впутать», <span class='clSrc'>ciągnąć</span> «тянуть» - <span class='clSrc'>wciągnąć</span> «втянуть»;</li>
<li><span class='clSrc'>wz-</span>: <span class='clSrc'>nieść</span> - <span class='clSrc'>wznieść</span>, <span class='clSrc'>bić</span> - <span class='clSrc'>wzbić</span>;</li>
<li><span class='clSrc'>wy-</span>: <span class='clSrc'>mówić</span> - <span class='clSrc'>wymówić</span>, <span class='clSrc'>palić</span> - <span class='clSrc'>wypalić</span>;</li>
<li><span class='clSrc'>do-</span>: <span class='clSrc'>żyć</span> - <span class='clSrc'>doźyć</span>, <span class='clSrc'>jeść</span> - <span class='clSrc'>dojeść</span>;</li>
<li><span class='clSrc'>za-</span>: <span class='clSrc'>lśnić</span> «сверкать» - <span class='clSrc'>zalśnic</span> «засверкать», <span class='clSrc'>wędrować</span> «странствовать» - <span class='clSrc'>zawędrować</span> «забрести»;</li>
<li><span class='clSrc'>z-</span>: <span class='clSrc'>moknąć</span> - zmoknąć;</li>
<li><span class='clSrc'>na-</span>: <span class='clSrc'>brać</span> - <span class='clSrc'>nabrać</span>, <span class='clSrc'>darować</span> - <span class='clSrc'>nadarować</span>;</li>
<li><span class='clSrc'>nad-</span>: <span class='clSrc'>pruć</span> «пороть» - <span class='clSrc'>nadpruć</span>, <span class='clSrc'>budować</span> - <span class='clSrc'>nadbudować</span>;</li>
<li><span class='clSrc'>o-</span>: <span class='clSrc'>świecić</span> - <span class='clSrc'>oświecić</span>, <span class='clSrc'>siodłać</span> «cедлать» - <span class='clSrc'>osiodłać</span>;</li>
<li><span class='clSrc'>od-</span>: <span class='clSrc'>chylić</span> - <span class='clSrc'>odchylić</span>, <span class='clSrc'>karmić</span> - <span class='clSrc'>odkarmić</span>;</li>
<li><span class='clSrc'>prze-</span>: <span class='clSrc'>płacić</span> - <span class='clSrc'>przepłacić</span>; эта приставка соответствует русским <span class='clSrc'>пере-</span> и <span class='clSrc'>про-</span>: «проводить» - <span class='clSrc'>przeprowadzić</span>, «переписывать» - <span class='clSrc'>przepisywać</span>;</li>
<li><span class='clSrc'>po-</span>: <span class='clSrc'>prosić</span> - <span class='clSrc'>poprosić</span>, <span class='clSrc'>skakać</span> - <span class='clSrc'>poskakać</span>;</li>
<li><span class='clSrc'>pod-</span>: <span class='clSrc'>strzyc</span> - <span class='clSrc'>podstrzyc</span>, <span class='clSrc'>twierdzić</span> - <span class='clSrc'>potwierdzić</span>;</li>
<li><span class='clSrc'>przy-</span>: <span class='clSrc'>szyć</span> - <span class='clSrc'>przyszyć</span>, <span class='clSrc'>klapnąć</span> «хлопнуть» - <span class='clSrc'>przyklapnąć</span>;</li>
<li><span class='clSrc'>roz-</span>: <span class='clSrc'>myśleć</span> - <span class='clSrc'>rozmyśleć</span> «раздумать», <span class='clSrc'>kraść</span> - <span class='clSrc'>rozkraść</span>;</li>
<li><span class='clSrc'>u-</span>: <span class='clSrc'>wodzić</span> - <span class='clSrc'>uwodzić</span>, <span class='clSrc'>zbierać</span> - <span class='clSrc'>uzbierać</span> «скопить».</li>
</ul>
