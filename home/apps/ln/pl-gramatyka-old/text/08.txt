
=== Правописание ch - h

<h1>Правописание ch - h</h1>
<p>Заднеязычный звук [x] может обозначаться в польском языке буквой <span class='clSrc'>h</span> или диграфом <span class='clSrc'>ch</span>. Диграф <span class='clSrc'>ch</span> пишется:</p>
<p><span class='clSrc'>а)</span> когда возможно чередование [x/š], например: <span class='clSrc'>mucha</span> - <span class='clSrc'>muszka</span>, <span class='clSrc'>suchy</span> - <span class='clSrc'>suszyć</span>, <span class='clSrc'>duch</span> - <span class='clSrc'>dusza</span>, <span class='clSrc'>piechota</span> - <span class='clSrc'>pieszo</span>;</p>
<p><span class='clSrc'>б)</span> в конце слова, например: <span class='clSrc'>mech</span>, <span class='clSrc'>ruch</span>, <span class='clSrc'>w oczach</span>, <span class='clSrc'>strach</span>, <span class='clSrc'>gmach</span> (исключения: <span class='clSrc'>Boh</span>, <span class='clSrc'>druh</span>);</p>
<p><span class='clSrc'>в)</span> после буквы <span class='clSrc'>s</span>, например: <span class='clSrc'>pascha</span>, <span class='clSrc'>schemat</span>, <span class='clSrc'>scholastyka</span>, <span class='clSrc'>schudnąć</span>;</p>
<p><span class='clSrc'>г)</span> в заимствованных словах, если в языке-источнике пишется <span class='clSrc'>ch</span>, например: <span class='clSrc'>charakter</span>, <span class='clSrc'>chaotyczny</span>, <span class='clSrc'>charytatywny</span>, <span class='clSrc'>chemia</span>.</p>
<p>Буква <span class='clSrc'>h</span> пишется:</p>
<p><span class='clSrc'>а)</span> если возможны чередования [x/g], [x/z], [x/ž], [x/dz], например: <span class='clSrc'>wahać się</span> - <span class='clSrc'>waga</span>, <span class='clSrc'>ważyć</span>, <span class='clSrc'>druh</span> - <span class='clSrc'>drużyna</span>, <span class='clSrc'>błahy</span> - <span class='clSrc'>błazen</span>, <span class='clSrc'>wataha</span> - <span class='clSrc'>watadze</span>;</p>
<p><span class='clSrc'>б)</span> если существуют вариантные формы слов без начального [x], например: <span class='clSrc'>harfa</span> или <span class='clSrc'>arfa</span>, <span class='clSrc'>Hanusia</span> или <span class='clSrc'>Anusia</span>, <span class='clSrc'>Hanna</span> или <span class='clSrc'>Anna</span>, <span class='clSrc'>Hindus</span> или <span class='clSrc'>Indus</span>;</p>
<p><span class='clSrc'>в)</span> в словах, которые начинаются с <span class='clSrc'>hiper-</span>, <span class='clSrc'>hipo-</span>, например: <span class='clSrc'>hiperbola</span>, <span class='clSrc'>hipochondria</span>, <span class='clSrc'>hipoteka</span> и др.;</p>
<p><span class='clSrc'>г)</span> если польскому звуку [x] в русском соответствует [g] или отсутствие звука, например:</p>
<ul>
<li><span class='clSrc'>historia</span> - история</li>
<li><span class='clSrc'>honor</span> - гонор</li>
<li><span class='clSrc'>hamak</span> - гамак</li>
<li><span class='clSrc'>higiena</span> - гигиена</li>
<li><span class='clSrc'>hołota</span> - голота</li>
<li><span class='clSrc'>hrabia</span> - граф</li>
<li><span class='clSrc'>hulać</span> - гулять</li>
<li><span class='clSrc'>alkohol</span> - алкоголь</li>
<li><span class='clSrc'>nahajka</span> - нагайка</li>
<li><span class='clSrc'>Bohdan</span> - Богдан</li>
<li><span class='clSrc'>Himalaje</span> - Гималаи</li>
<li><span class='clSrc'>Mahomet</span> - Магомет</li>
<li><span class='clSrc'>Holandia</span> - Голландия</li>
<li><span class='clSrc'>huragan</span> - ураган</li>
<li><span class='clSrc'>gehenna</span> - геена</li>
<li><span class='clSrc'>Hipolit</span> - Ипполит</li>
</ul>
<p>Исключение: <span class='clSrc'>hokej</span> - хоккей.</p>
