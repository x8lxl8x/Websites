
=== Способы выражения средства действия

<h1>Способы выражения средства действия</h1>
<ul>
<li><span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>pisać piórem</span> «писать пером». Польские конструкции этого типа более регулярны, чем соответствующие русские, они активное используются вместе с глаголами движения: <span class='clSrc'>jechać tramwajem</span> «ехать на трамвае», <span class='clSrc'>przyjechać Mercedesem</span> «приехать на «Мерседесе»», <span class='clSrc'>wybrać się do kogoś autobusem</span> «выбраться к кому-л. на автобусе», <span class='clSrc'>lądować statkiem</span> «причалить на корабле», <span class='clSrc'>wrócić siedemnastką</span> «вернуться автобусом (троллейбусом, трамваем) маршрута № 17», <span class='clSrc'>zjechać windą</span> «спуститься в лифте», <span class='clSrc'>pojechać saniami</span> «поехать на санях»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>na własne oczy</span> «своими глазами», <span class='clSrc'>nie widzi na lewe oko</span> «не видеть левым глазом»;</li>
<li><span class='clSrc'>przez</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>ogłosić przez radio</span> «объявить по радио», <span class='clSrc'>mówić przez telefon</span> «говорить по телефону»;</li>
<li><span class='clSrc'>przy</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>pracować przy łopatach</span> «работать лопатами»;</li>
<li><span class='clSrc'>w</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>szyć w rękach</span> «шить руками».</li>
</ul>
