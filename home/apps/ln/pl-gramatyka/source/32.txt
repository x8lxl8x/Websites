
=== Степени сравнения прилагательных

<h1>Степени сравнения прилагательных</h1>
<p>Формы степеней сравнения образуют качественные прилагательные. Сравнительная степень образуется при помощи двух суффиксов. Первый суффикс <span class='clSrc'>-sz-</span> добавляется в случае, если основа заканчивается на одиночный согласный (<span class='clSrc'>śmiały</span> «смелый» - <em>śmiel<span class='clSrc'>sz</span>y</em>, <span class='clSrc'>stary</span> «старый» - <em>star<span class='clSrc'>sz</span>y</em>). У прилагательных с суффиксами <span class='clSrc'>-k-, -ek-, -ok-</span> сравнительная степень образуется непосредственно от корня, суффикс исчезает (<span class='clSrc'>brzydki</span> «некрасивый» - <em>brzydszy</em>, <span class='clSrc'>wysoki</span> «высокий» - <em>wyższy</em>). При этом могут происходить следующие чередования:</p>
<table>
<tr>
<td><span class='clSrc'>s/ż</span></td>
<td><span class='clSrc'>wysoki</span> «высокий» - <span class='clSrc'>wyższy</span></td>
</tr>
<tr>
<td><span class='clSrc'>g/ż</span></td>
<td><span class='clSrc'>drogi</span> «дорогой» - <span class='clSrc'>droższy</span></td>
</tr>
<tr>
<td><span class='clSrc'>ł/l</span></td>
<td><span class='clSrc'>wesoły</span> «веселый» - <span class='clSrc'>weselszy</span></td>
</tr>
</table>
<p>Возможно также чередование гласных:</p>
<table>
<tr>
<td><span class='clSrc'>а/e</span></td>
<td><span class='clSrc'>śmiały</span> «смелый» - <span class='clSrc'>śmielszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>o/e</span></td>
<td><span class='clSrc'>wesoły</span> «веселый» - <span class='clSrc'>weselszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>ą/ę</span></td>
<td><span class='clSrc'>wąski</span> «узкий» - <span class='clSrc'>węższy</span></td>
</tr>
</table>
<p>Второй суффикс <span class='clSrc'>-ejsz-</span> употребляется при стечении согласных в конце основы (<span class='clSrc'>ciepły</span> «теплый» - <em>ciepl<span class='clSrc'>ejsz</span>y</em>, <span class='clSrc'>uprzejmy</span> «вежливый» - <em>uprzejmi<span class='clSrc'>ejsz</span>y</em>). При этом также происходят следующие чередования:</p>
<table>
<tr>
<td><span class='clSrc'>w/w'</span></td>
<td><span class='clSrc'>łatwy</span> «легкий» - <span class='clSrc'>łatwiejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>m/m'</span></td>
<td><span class='clSrc'>uprzejmy</span> «вежливый» - <span class='clSrc'>uprzejmiejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>ł/l</span></td>
<td><span class='clSrc'>szczupły</span> «худой» - <span class='clSrc'>szczuplejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>n/n'</span></td>
<td><span class='clSrc'>silny</span> «сильный» - <span class='clSrc'>silniejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>r/rz</span></td>
<td><span class='clSrc'>mądry</span> «умный» - <span class='clSrc'>mądrzejszy (mędrszy)</span></td>
</tr>
<tr>
<td><span class='clSrc'>st/ść</span></td>
<td><span class='clSrc'>gęsty</span> «густой» - <span class='clSrc'>gęściejszy (gęstszy)</span></td>
</tr>
<tr>
<td><span class='clSrc'>sł/śl</span></td>
<td><span class='clSrc'>ścisły</span> «точный» - <span class='clSrc'>ściślejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>sn/śń</span></td>
<td><span class='clSrc'>nowoczesny</span> «современный» - <span class='clSrc'>nowocześniejszy</span></td>
</tr>
<tr>
<td><span class='clSrc'>zn/źń</span></td>
<td><span class='clSrc'>żyzny</span> «урожайный» - <span class='clSrc'>żyźniejszy</span></td>
</tr>
</table>
<p>Отдельная группа прилагательных образует формы от другой основы: <span class='clSrc'>dobry</span> «хороший» - <em>lepszy</em>; <span class='clSrc'>zły</span> «плохой» - <em>gorszy</em>; <span class='clSrc'>mały</span> «маленький» - <em>mniejszy</em>; <span class='clSrc'>wielki, duży</span> «большой» - <em>większy</em>; <span class='clSrc'>lekki</span> «легкий» - <em>lżejszy</em>.</p>
<blockquote>
<p>Формы сравнительной степени склоняются по образцу прилагательного с соответствующей основой: <span class='clSrc'>widziałem człowieka starszego od mojego ojca</span> «я видел человека старше моего отца». В именительном падеже прилагательные в этой форме, согласующиеся с лично-мужскими формами существительных, образуют лично-мужские формы: <span class='clSrc'>człowiek uprzejmiejszy od mojego kolegi</span> «человек вежливее моего коллеги» - <span class='clSrc'>ludzie uprzejmiejsi od moich kolegów</span>.</p>
</blockquote>
<p>Превосходная степень сравнения образуется от прилагательного в форме сравнительной степени при помощи приставки <span class='clSrc'>naj-</span> (<span class='clSrc'>stary</span> «старый» - <span class='clSrc'>starszy</span> - <span class='clSrc'>naj</span>starszy</em>; <span class='clSrc'>dobry</span> «хороший» - <span class='clSrc'>lepszy</span> - <span class='clSrc'>naj</span>lepszy</em>). Ее формы также изменяются по родам, числам, падежам.</p>
<p>Сложные формы степеней сравнения образуют прилагательные более позднего, чаще всего иноязычного, происхождения. Они образуются путем соединения прилагательного с наречиями <span class='clSrc'>bardziej</span> (для образования сравнительной степени) и <span class='clSrc'>najbardziej</span> (для образования превосходной степени): <span class='clSrc'>typowy</span> «типичный» - <em>bardziej typowy</em>, <span class='clSrc'>interesujący</span> «интересный» - <em>najbardziej interesujący</em>.</p>
