
=== Правописание заимствованных слов

<h1>Правописание заимствованных слов</h1>
<p>Для польского языка характерно сохранение оригинального написания заимствованных слов, в первую очередь - имен собственных: <span class='clSrc'>Schiller</span> «Шиллер», <span class='clSrc'>Bill Clinton</span> «Билл Клинтон», <span class='clSrc'>Strauss</span> «Штраус», <span class='clSrc'>Stuttgart</span> «Штутгарт», <span class='clSrc'>Schubert</span> «Шуберт», <span class='clSrc'>Chile</span> «Чили», <span class='clSrc'>Chikago</span> «Чикаго», <span class='clSrc'>Churchil</span> «Черчилль», <span class='clSrc'>Chomsky</span> «Хомский», <span class='clSrc'>Chagal</span> «Шагал», <span class='clSrc'>Chopin</span> «Шопен», <span class='clSrc'>Curie</span> «Кюри», <span class='clSrc'>Don Juan</span> «Дон Жуан», <span class='clSrc'>Jan Jakub Rousseau</span> «Жан-Жак Руссо», <span class='clSrc'>Goethe</span> «Гёте», <span class='clSrc'>Baumarchais</span> «Бомарше», <span class='clSrc'>Charles Chaplin</span> «Чарльз Чаплин» и др.</p>
<p>Сохраняют свое исходное написание и некоторые интернациональные термины: <span class='clSrc'>scherzo</span> «скерцо», <span class='clSrc'>intermezzo</span> «интермеццо», <span class='clSrc'>staccato</span> «стакатто», <span class='clSrc'>démarche</span> «демарш», <span class='clSrc'>résumé</span> «резюме», <span class='clSrc'>en face</span> «анфас», <span class='clSrc'>charleston</span> «чарльстон», <span class='clSrc'>jazz</span> «джаз», <span class='clSrc'>judo</span> «дзюдо», <span class='clSrc'>junta</span> «хунта», <span class='clSrc'>hall</span> «холл», <span class='clSrc'>hobby</span> «хобби» и др.</p>
<p>Некоторые часто употребляемые заимствованные слова получают полонизированное графическое оформление, например: <span class='clSrc'>makijaż</span> «макияж», <span class='clSrc'>komputer</span> «комьютер», <span class='clSrc'>biznes</span> «бизнес», <span class='clSrc'>lider</span> «лидер», <span class='clSrc'>dżem</span> «джем», <span class='clSrc'>kowboj</span> «ковбой», <span class='clSrc'>Paryż</span> «Париж», <span class='clSrc'>Londyn</span> «Лондон», <span class='clSrc'>Rzym</span> «Рим» и т.п.</p>
<p>Некоторые слова допускают двоякое - оригинальное и адаптированное написание, ср.: <span class='clSrc'>wademekum</span> - <span class='clSrc'>vademecum</span> «справочник», <span class='clSrc'>weto</span> - <span class='clSrc'>veto</span> «вето», <span class='clSrc'>wiledżiatura</span> - <span class='clSrc'>wilegiatura</span> «дача», <span class='clSrc'>adadżio</span> - <span class='clSrc'>adagio</span> «адажио».</p>
<p>При передаче на письме слов, которые в оригинале написаны кирилицей, обычно используется транслитерация - передача графических знаков одного языка их эквивалентами в другом языке, например: <span class='clSrc'>F. Skarynа</span> «Ф. Скарына», <span class='clSrc'>Szyszkow</span> «Шишков», <span class='clSrc'>Katajew</span> «Катаев», <span class='clSrc'>Karatkiewicz</span> «Караткевіч», <span class='clSrc'>Maładzieczna</span> «Маладзечна», <span class='clSrc'>Słonim</span> «Слоним» и др.</p>
