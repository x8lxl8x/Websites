
=== Правописание ż - rz

<h1>Правописание ż - rz</h1>
<p>Буква <span class='clSrc'>ż</span> и диграф <span class='clSrc'>rz</span> обозначают один и тот же звук [ž] (при оглушении - звук [š]), различие же их написания, как и в случае букв <span class='clSrc'>ó</span> - <span class='clSrc'>u</span>, обусловлено особенностями исторического развития польского языка.</p>
<p>Буква <span class='clSrc'>ż</span> обозначает [ž] общеславянского происхождения, ее эквивалентом в русском языке выступает буква <span class='clSrc'>ж</span>, ср.: <span class='clSrc'>żelazny</span> «железный», <span class='clSrc'>żona</span> «жена», <span class='clSrc'>ważny</span> «важный» и др. Правила, которым подчиняется написание буквы <span class='clSrc'>ż</span>, таковы:</p>
<p><span class='clSrc'>а)</span> пишем <span class='clSrc'>ż</span>, если в других формах того же слова или в словах с тем же корнем происходят чередования [ž/g], [ž/dz], [ž/z], [ž/s], [ž/ź], [ž/h], например:</p>
<ul>
<li>trwożnie - trwoga</li>
<li>pieniążek - pieniądz</li>
<li>grożę - groza</li>
<li>obrażać - obraźliwy</li>
<li>narożnik - na rogu</li>
<li>drużyna - druh</li>
<li>mrożenie - mróz</li>
<li>podwyższenie - wysoko</li>
<li>przerażenie - przerazić</li>
<li>zwężać - wąsko</li>
</ul>
<p><span class='clSrc'>б)</span> в словах, которые заканчиваются на <span class='clSrc'>-aż</span>, <span class='clSrc'>-eż</span>, например: <span class='clSrc'>sprzedaż</span> «продажа», <span class='clSrc'>bandaż</span> «бандаж», <span class='clSrc'>witraż</span> «витраж», <span class='clSrc'>młodzież</span> «молодежь», <span class='clSrc'>odzież</span> «одежда», <span class='clSrc'>grabież</span> «грабеж»;</p>
<p><span class='clSrc'>в)</span> в частице <span class='clSrc'>-że</span>, <span class='clSrc'>-ż</span>, например: <span class='clSrc'>tenże</span> «этот же», <span class='clSrc'>bądźże</span> «будь же», <span class='clSrc'>stańże</span> «стань же», <span class='clSrc'>cóż</span> «што ж», <span class='clSrc'>czyż</span> «разве ж», <span class='clSrc'>gdyż</span> «если же»;</p>
<p><span class='clSrc'>г)</span> в заимствованных словах после буквы <span class='clSrc'>n</span>, например: <span class='clSrc'>aranżować</span> «организовывать: аранжировать», <span class='clSrc'>branża</span> «область (промышленности или торговли)», <span class='clSrc'>oranżada</span> «апельсиновый охлаждающий напиток», <span class='clSrc'>rewanż</span> «реванш».</p>
<p>Диграф <span class='clSrc'>rz</span> пишется в соответствии с правилами:</p>
<p><span class='clSrc'>а)</span> когда в разных формах одного и того же слова или в однокорневых словах происходит чередование [ž/r], например:</p>
<ul>
<li>reformatorzy - reformator</li>
<li>w literaturze - literatura</li>
<li>w komputerze - komputer</li>
<li>tworzyć - twórca</li>
<li>powtórzyć - powtórka</li>
<li>wzorzec - wzorca</li>
<li>o charakterze - charakter</li>
<li>radarzysta - radar</li>
<li>dworzanin - dwór</li>
<li>maturzysta - matura</li>
<li>dworzec - na dworcu</li>
<li>lekarz - lekarstwo</li>
<li>ślusarz - ślusarski</li>
<li>harcerz - harcerstwo</li>
</ul>
<p><span class='clSrc'>б)</span> когда в восточнославянских языках польскому звуку [ž] соответствует [r] или [r'] (это касается и тех польских слов, которые не допускают чередования [ž/r]), например:</p>
<ul>
<li><span class='clSrc'>rzeka</span> - река</li>
<li><span class='clSrc'>rzadko</span> - редко</li>
<li><span class='clSrc'>orzech</span> - орех</li>
<li><span class='clSrc'>korzeń</span> - корень</li>
<li><span class='clSrc'>porządek</span> - порядок</li>
<li><span class='clSrc'>rząd</span> - ряд</li>
<li><span class='clSrc'>rzemień</span> - ремень</li>
<li><span class='clSrc'>rzemiosło</span> - ремесло</li>
<li><span class='clSrc'>Rzym</span> - Рим</li>
<li><span class='clSrc'>wierzba</span> - верба</li>
<li><span class='clSrc'>wierzch</span> - верх</li>
</ul>
<p><span class='clSrc'>в)</span> после букв, которые обозначают согласные [v], [b], [p], [d], [t], [g], [k], [x], например:</p>
<ul>
<li><span class='clSrc'>wyprzedaż</span> - распродажа</li>
<li><span class='clSrc'>potrzeba</span> - потребность</li>
<li><span class='clSrc'>chrześcijański</span> - христианский</li>
<li><span class='clSrc'>wrzątek</span> - кипяток</li>
<li><span class='clSrc'>przekonanie</span> - убеждение</li>
<li><span class='clSrc'>wytrzymać</span> - выдержать</li>
<li><span class='clSrc'>skrzydło</span> - крыло</li>
<li><span class='clSrc'>brzmienie</span> - звучание</li>
<li><span class='clSrc'>wydrzeć</span> - вырвать</li>
<li><span class='clSrc'>ugrzęznąć</span> - погрязнуть</li>
</ul>
<blockquote>
<p>Исключения:</p>
<ul>
<li>слова <span class='clSrc'>pszenica</span> «пшеница», <span class='clSrc'>pszczoła</span> «пчела», <span class='clSrc'>kształt</span> «форма», <span class='clSrc'>bukszpan</span> «самшит», <span class='clSrc'>wszystko</span> «всё», <span class='clSrc'>wszędzie</span> «везде», <span class='clSrc'>zawsze</span> «всегда» и др.;</li>
<li>формы сравнительной степени прилагательных: <span class='clSrc'>głupszy</span> «более глупый», <span class='clSrc'>krótszy</span> «более короткий», <span class='clSrc'>rzadszy</span> «более редкий»;</li>
<li>слова с буквой <span class='clSrc'>ż</span> в начале корня после приставок <span class='clSrc'>ob-</span>, <span class='clSrc'>od-</span>, <span class='clSrc'>pod-</span>, например: <span class='clSrc'>obżerać się</span> «объедаться», <span class='clSrc'>odżałować</span> «оплакать; не поскупиться», <span class='clSrc'>odżyć</span> «ожить», <span class='clSrc'>nadżółkły</span> «пожелтевший»;</li>
<li>слова, которые заканчиваются на <span class='clSrc'>-że</span>, например: <span class='clSrc'>dajże</span> «дай же», <span class='clSrc'>zróbże</span> «сделай же», <span class='clSrc'>także</span> «также», <span class='clSrc'>skądże</span> «откуда же».</li>
</ul>
</blockquote>
<p><span class='clSrc'>г)</span> в словах, которые заканчиваются на <span class='clSrc'>-mierz</span> и <span class='clSrc'>-mistrz</span>, например: <span class='clSrc'>Kazimierz</span>, <span class='clSrc'>Włodzimierz</span>, <span class='clSrc'>sztukmistrz</span>, <span class='clSrc'>zegarmistrz</span>, <span class='clSrc'>kuchmistrz</span>.</p>
