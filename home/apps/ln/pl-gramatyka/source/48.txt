
=== Причастия и деепричастия

<h1>Причастия и деепричастия</h1>
<p>В польском языке существуют следующие причастия и деепричастия:</p>
<p>Причастия</p>
<ul>
<li>действительного залога</li>
<li>страдательного залога</li>
</ul>
<p>Деепричастия</p>
<ul>
<li>несовершенного вида</li>
<li>совершенного вида</li>
</ul>
<h2>Причастие действительного залога</h2>
<p>Форма, аналогичная русскому причастию действительного залога настоящего времени. Образуется только от глаголов несовершенного вида. Эту форму составляют следующие компоненты:</p>
<ol>
<li>основа настоящего времени (3-го лица множественного числа),</li>
<li>суффикс <span class='clSrc'>-ąc</span>,</li>
<li>падежно-родовые окончания прилагательного: <span class='clSrc'>czytać</span> - <span class='clSrc'>czytaj-ą</span> - <span class='clSrc'>czytaj-ąc-y</span>; <span class='clSrc'>brać</span> - <span class='clSrc'>bior-ą</span> - <span class='clSrc'>bior-ąc-y</span>.</li>
</ol>
<h2>Причастие страдательного залога</h2>
<p>Эта форма похожа на форму русского причастия страдательного залога прошедшего времени. Склоняется по образцу прилагательного. Образуется при помощи суффиксов <span class='clSrc'>-n-, -on-, -t-</span>. Суффикс <span class='clSrc'>-n-</span> характерен для инфинитива, основа которого заканчивается на следующие гласные:</p>
<p>1) <span class='clSrc'>-а-</span>; суффикс присоединяется к основе прошедшего времени 3-го лица единственного числа (<span class='clSrc'>czytać</span> - <span class='clSrc'>czytał</span> - <em>czyta<span class='clSrc'>n</span>y</em>, <span class='clSrc'>dać</span> - <span class='clSrc'>dał</span> - <em>da<span class='clSrc'>n</span>y</em>);</p>
<p>2) <span class='clSrc'>-е-</span>; суффикс присоединяется к основе прошедшего времени 3-го лица единственного числа, в этой основе происходит чередование <span class='clSrc'>е/a</span> (<span class='clSrc'>widzieć</span> - <span class='clSrc'>widział</span> - <em>widzia<span class='clSrc'>n</span>y</em>).</p>
<p>Суффикс <span class='clSrc'>-on-</span> характерен для инфинитива, основа которого заканчивается на следющие звуки:</p>
<p>1) <span class='clSrc'>-i-, -y-</span> (II спряжение); присоединяется к основе 3-го лица множественного числа: <span class='clSrc'>suszyć</span> - <span class='clSrc'>suszą</span> - <em>susz<span class='clSrc'>on</span>y</em>, <span class='clSrc'>nosić</span> - <span class='clSrc'>noszą</span> - <em>nosz<span class='clSrc'>on</span>y</em>;</p>
<p>2) согласный (<span class='clSrc'>ś, ź</span>); присоединяется к основе настоящего времени 3-го лица единственного числа: <span class='clSrc'>nieść</span> - <span class='clSrc'>niesie</span> - <em>niesi<span class='clSrc'>on</span>y</em>.</p>
<p>Тот же суффикс используется для образования причастия от инфинитива, который заканчивается на <span class='clSrc'>-с</span>: <span class='clSrc'>piec</span> - <span class='clSrc'>piecze</span> - <em>piecz<span class='clSrc'>on</span>y</em>.</p>
<p>При склонении в лично-мужских формах, образованных при помощи этого суффикса, могут происходить следующие чередования:</p>
<p>а) суффиксальный гласный <span class='clSrc'>o</span> чередуется с <span class='clSrc'>е</span>: <em>chwal<span class='clSrc'>on</span>y</em> «хвалёный» - <em>chwal<span class='clSrc'>en</span>i</em>;</p>
<p>б) основа на <span class='clSrc'>ć, dź</span> становится твердой; <span class='clSrc'>ć/c, dź/dz</span> (<span class='clSrc'>zgnieść</span> «смять» - <span class='clSrc'>zgniotę</span> - <em>zgnie<span class='clSrc'>c</span>iesz</em> - <em>zgnie<span class='clSrc'>c</span>iony</em> - <em>zgnie<span class='clSrc'>c</span>eni</em>, <span class='clSrc'>wieść</span> - <span class='clSrc'>wiodę</span> - <em>wie<span class='clSrc'>dz</span>iesz</em> - <em>wie<span class='clSrc'>dz</span>iony</em> - <em>wie<span class='clSrc'>dz</span>eni</em>).</p>
<p>Cуффикс <span class='clSrc'>-t-</span> характерен для инфинитива, который заканчивается следующим образом:</p>
<p>1) на <span class='clSrc'>y, i, u, ą</span>, принадлежащие корню; присоединяется к основе прошедшего времени (лично-мужская форма 3-го лица множественного числа): <span class='clSrc'>kryć</span> - <span class='clSrc'>kryli</span> - <em>kry<span class='clSrc'>t</span>y</em>; <span class='clSrc'>bić</span> - <span class='clSrc'>bili</span> - <em>bi<span class='clSrc'>t</span>y</em>; <span class='clSrc'>kłuć</span> «колоть» - <span class='clSrc'>kłuli</span> - <em>kłu<span class='clSrc'>t</span>y</em>; <span class='clSrc'>ciąć</span> «резать» - <span class='clSrc'>cięli</span> - <em>cię<span class='clSrc'>t</span>y</em>;</p>
<p>2) на <span class='clSrc'>-rże-, -rze-</span>; также присоединяется к основе прошедшего времени (лично-мужская форма 3-го лица множественного числа: <span class='clSrc'>drzeć</span> «драть» - <span class='clSrc'>darli</span> - <em>dar<span class='clSrc'>t</span>y</em>; <span class='clSrc'>wytrzeć</span> «вытереть» - <span class='clSrc'>wytarli</span> - <em>wytar<span class='clSrc'>t</span>y</em>;</p>
<p>3) на <span class='clSrc'>-ną-</span>; при присоединении суффикса <span class='clSrc'>-t-</span> к основе <span class='clSrc'>-ną-</span> меняется на <span class='clSrc'>-nię-</span> (<span class='clSrc'>zamknąć</span> - <em>zamk<span class='clSrc'>nię</span>ty</em>; <span class='clSrc'>ciągnąć</span> - <em>ciąg<span class='clSrc'>nię</span>ty</em>); искл. - <span class='clSrc'>natchnąć</span> - <em>natchniony</em>.</p>
<h2>Деепричастие несовершенного вида</h2>
<p>Образуется от основы настоящего времени глаголов несовершенного вида при помощи суффикса <span class='clSrc'>-ąc-</span> (по аналогии с причастиями действительного залога, см. выше) (<span class='clSrc'>widzeć</span> - <em>widząc</em>, <span class='clSrc'>pić</span> - <em>pijąc</em>, <span class='clSrc'>móc</span> «мочь» - <em>mogąc</em>).</p>
<p>Это деепричастие обозначает действие, одновременное с основным, выраженным сказуемым (<span class='clSrc'>Patrząc na te piękne kwiaty myślę o wiośnie</span> «глядя на эти красивые цветы, я думаю о весне»; <span class='clSrc'>Leżałem na plaży czytając gazety</span> «я лежал на пляже, читая газеты»).</p>
<h2>Деепричастие совершенного вида</h2>
<p>Образуется от основы прошедшего времени глаголов совершенного вида 3-го лица единственного числа мужского рода при помощи суффиксов <span class='clSrc'>-wszy</span> и <span class='clSrc'>-łszy</span>:</p>
<p>1) cуффикс <span class='clSrc'>-wszy</span> присоединяется к основе на гласный (<span class='clSrc'>włożyć</span> «надеть» - <em>włoży<span class='clSrc'>wszy</span>; <span class='clSrc'>zapomnieć</span> «забыть» - <span class='clSrc'>zapomniał</span> - <em>zapomnia<span class='clSrc'>wszy</span>);</p>
<p>2) суффикс <span class='clSrc'>-łszy-</span> (читается без <span class='clSrc'>ł</span>) присоединяется к основе на согласный (<span class='clSrc'>dowieść</span> «доказать» - <em>dowiód<span class='clSrc'>łszy</span>, <span class='clSrc'>móc</span> - <span class='clSrc'>mógł</span> - <em>móg<span class='clSrc'>łszy</span>).</p>
<p>Эти деепричастия обозначают действие, предшествующее основному, названному сказуемым (<span class='clSrc'>Wziąwszy parasol wyszedłem na ulicę</span> «Взяв зонт, я вышел на улицу»).</p>
