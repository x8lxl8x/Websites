
=== Изъявительное наклонение. Настоящее время

<h1>Изъявительное наклонение. Настоящее время</h1>
<h2>I спряжение</h2>
<p>Глаголы с окончаниями <span class='clSrc'>-ę</span>, <span class='clSrc'>-esz</span> в 1-м и 2-м лице единственного числа. Они имеют инфинитив на на <span class='clSrc'>-c</span> (<span class='clSrc'>piec</span>, <span class='clSrc'>biec</span>, <span class='clSrc'>strzec</span>), а также инфинитив, который заканчивается на:</p>
<ol>
<li><span class='clSrc'>согласный</span> + <span class='clSrc'>ć</span> (<span class='clSrc'>nieść</span>, <span class='clSrc'>wiezć</span>);</li>
<li><span class='clSrc'>-ować</span> (<span class='clSrc'>rysować</span>, <span class='clSrc'>rachować</span> «считать»);</li>
<li><span class='clSrc'>-ąć, -nąć</span> (<span class='clSrc'>dąć</span>, <span class='clSrc'>dźwignąć</span> «поднять»);</li>
<li><span class='clSrc'>-ić, -yć, -uć</span> (<span class='clSrc'>pić</span>, <span class='clSrc'>żyć</span>, <span class='clSrc'>wyżyć</span>, <span class='clSrc'>pluć</span> «плевать») - односложные и производные от них;</li>
<li><span class='clSrc'>-ać</span> (<span class='clSrc'>pisać</span>) (частично);</li>
<li><span class='clSrc'>-eć</span> (<span class='clSrc'>chcieć</span>) (частично).</li>
</ol>
<p>Для более точного выделения глаголов этого типа следует ориентироваться на однокоренные глаголы в русском и белорусском языке (<em>брать - беру, берешь</em> / <span class='clSrc'>brać - biorę, bierzesz</span>).</p>

<p>Глаголы с суффиксом инфинитива <span class='clSrc'>-ować, -ywać</span> заменяют этот суффикс на <span class='clSrc'>uj-</span> (<span class='clSrc'>rysować</span> - <em>rys<span class='clSrc'>uj</span>ę</em>, <span class='clSrc'>pisywać</span> - <em>pis<span class='clSrc'>uj</span>ę</em>). Есть и вариантные формы (см. ниже).</p>
<h2>II спряжение</h2>
<p>Глаголы с окончаниями <span class='clSrc'>-ę, -isz/-ysz</span> в 1-м и 2-м лице единственного числа. Инфинитивы таких глаголов заканчиваются на:</p>
<ol>
<li><span class='clSrc'>-ić/-yć</span> (<span class='clSrc'>mówić</span> «говорить», <span class='clSrc'>tańczyć</span> «танцевать») (двусложные);</li>
<li><span class='clSrc'>-ać</span> (<span class='clSrc'>spać</span>) (частично);</li>
<li><span class='clSrc'>-eć</span> (<span class='clSrc'>widzieć</span>, <span class='clSrc'>słyszeć</span>) (частично).</li>
</ol>
<p>Для более точного выделения глаголов этого типа следует ориентироваться на однокоренные глаголы в русском и белорусском языке (<em>видеть - видишь</em> / <span class='clSrc'>widzieć - widzę, widzisz</span>).</p>

<p>Гласный <span class='clSrc'>-у-</span> употребляется после затвердевших согласных, <span class='clSrc'>-i-</span> - после мягких.</p>
<p>Чередования гласных в этом типе спряжения отсутствуют, но встречаются чередования согласных.</p>
<p>Если основа в 1-м лице единственного числа заканчивается на затвердевший согласный, ее твердость сохраняется в 3-м лице множественного числа, во всех же остальных формах она смягчается. При этом имеют место следующие чередования согласных:</p>

<p>Но есть случаи, когда затвердевшие согласные остаются без изменения:</p>
<ul>

<li><span class='clSrc'>sz</span>: <span class='clSrc'>płoszę</span> «пугаю» - <span class='clSrc'>płoszysz</span>, <span class='clSrc'>ruszę</span> «отправлюсь» - <span class='clSrc'>ruszysz</span>, <span class='clSrc'>suszę</span> - <span class='clSrc'>suszysz</span>;</li>
<li><span class='clSrc'>ż</span>: <span class='clSrc'>wróżę</span> «ворожу» - <span class='clSrc'>wróżysz</span>, <span class='clSrc'>służę</span> - <span class='clSrc'>służysz</span>;</li>
<li><span class='clSrc'>szcz</span>: <span class='clSrc'>wrzeszczę</span> «кричу» - <span class='clSrc'>wrzeszczysz</span>, <span class='clSrc'>piszczę</span> - <span class='clSrc'>piszczysz</span>;</li>
<li><span class='clSrc'>żdż</span>: <span class='clSrc'>miażdżę</span> «давлю» - <span class='clSrc'>miażdżysz</span>;</li>
<li><span class='clSrc'>rz</span>: <span class='clSrc'>chmurzę</span> - <span class='clSrc'>chmurzysz</span>.</li>
</ul>
<h2>III спряжение</h2>
<p>Глаголы, которые в 1-м и 2-м лице единственного числа имеют окончания <span class='clSrc'>-am, -asz</span> и <span class='clSrc'>-em, -esz</span>. Инфинитив заканчивается на <span class='clSrc'>-ać</span> (<span class='clSrc'>słuchać</span>), <span class='clSrc'>-eć</span> (<span class='clSrc'>rozumieć</span>).</p>
<p>При выделении глаголов этого типа следует ориентироваться на однокоренные глаголы в русском и белорусском языке. Здесь им соответствуют формы на <span class='clSrc'>-аю, -аешь, -ею, -еешь</span>, т.е. те, которые наращивают <span class='clSrc'>j</span> после основы перед окончанием: <em>читать - читаю, читаешь</em> / <span class='clSrc'>czytać - czytam, czytasz</span>. В польских глаголах <span class='clSrc'>j</span> проявляется только в форме 3-го лица множественного числа.</p>
