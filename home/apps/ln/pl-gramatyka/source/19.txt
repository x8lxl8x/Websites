
=== Ударение

<h1>Ударение</h1>
<p>В польском языке ударение имеет постоянное место в слове. В большинстве случаев оно приходится на второй от конца слог: <span class='clSrc'>`konik</span> «лошадка», <span class='clSrc'>do`nośny</span> «звучный», <span class='clSrc'>niezapom`niany</span> «незабвенный». Постоянное ударение в польском языке имеет также большинство заимствованных слов: <span class='clSrc'>al`fabet</span> «алфавит», <span class='clSrc'>e`pizod</span> «эпизод», <span class='clSrc'>lo`teria</span> «лотерея», <span class='clSrc'>`student</span> «студент».</p>
<p>В многосложных словах, как собственно польских, так и заимствованных, кроме ударения на предпоследнем слоге, присутствует также ударение на первом слоге: <span class='clSrc'>`satys`fakcja</span> «удовлетворение», <span class='clSrc'>`rehabili`tacja</span> «реабилитация», <span class='clSrc'>`wiceprzewodni`czący</span> «заместитель председателя», <span class='clSrc'>`nieodpowie`dzialność</span> «безответственность», <span class='clSrc'>`nieprawo`rządny</span> «незаконный», <span class='clSrc'>`wieloczynno`ściowy</span> «многофункциональный», <span class='clSrc'>`przed`pokoik</span> «маленькая прихожая».</p>
<p>Но некоторые собственно польские и заимствованные слова не подчиняются этому правилу. Вот примеры исключений:</p>
<p>1) ударение может находиться на четвертом слоге от конца слова. Такое ударение имеют глагольные формы 1-го и 2-го лица условного наклонения: <span class='clSrc'>poje`chalibyśmy</span> «мы бы поехали», <span class='clSrc'>`dalibyście</span> «вы бы дали»;</p>
<p>2) ударение может приходиться на третий от конца слог. Такое ударение имеют следующие слова:</p>
<ul>
<li>некоторые польские существительные: <span class='clSrc'>o`kolica</span> «околица», <span class='clSrc'>rzeczpo`spolita</span> «республика (чаще о Польше)»;</li>
<li>некоторые существительные, заимствованные из английского и немецкого языков или посредством их: <span class='clSrc'>re`pertuar</span> «репертуар», <span class='clSrc'>re`zerwuar</span> «резервуар», <span class='clSrc'>`dżentelmen</span> «джентельмен», <span class='clSrc'>`wunderkind</span> «вундеркинд»;</li>
<li>некоторые существительные, заимствованные из греческого и латинского языков: <span class='clSrc'>a`naliza</span> «анализ», <span class='clSrc'>hi`perbola</span> «гипербола», <span class='clSrc'>indy`widuum</span> «индивидуум», <span class='clSrc'>`leksykon</span> «лексикон», <span class='clSrc'>`maksimum</span> «максимум», <span class='clSrc'>`minimum</span> «минимум», <span class='clSrc'>uni`wersytet</span> «университет», <span class='clSrc'>`ryzyko</span> «риск», в том числе с суффиксами <span class='clSrc'>-yka/-ika</span>: <span class='clSrc'>gra`matyka</span> «грамматика», <span class='clSrc'>publi`cystyka</span> «публицистика», <span class='clSrc'>`grafika</span> «графика»;</li>
<li>некоторые формы существительных (<span class='clSrc'>`ogółem</span> «вообще», <span class='clSrc'>`szczegóły</span> «подробности»);</li>
<li>некоторые формы числительных, например: <span class='clSrc'>`czterysta</span> «четыреста», <span class='clSrc'>`siedemset</span> «семьсот»;</li>
<li>некоторые формы прошедшего времени и условного наклонения: <span class='clSrc'>`byliśmy</span> «мы были», <span class='clSrc'>zro`biliśmy</span> «мы сделали», <span class='clSrc'>`mieliby</span> «они бы имели», <span class='clSrc'>je`chałaby</span> «она бы ехала»;</li>
<li>четырехсложные формы прилагательного <span class='clSrc'>powinien</span> «должен»: <span class='clSrc'>po`winniśmy</span>, <span class='clSrc'>po`winnyśmy</span>, <span class='clSrc'>po`winniście</span>, <span class='clSrc'>po`winnyście</span>;</li>
<li>некоторые заимствованные собственные имена и географические названия <span class='clSrc'>Na`poleon</span>, <span class='clSrc'>`Cycero</span>, <span class='clSrc'>`Hannibal</span>, <span class='clSrc'>`Ferdynand</span>, <span class='clSrc'>`Romulus</span>, <span class='clSrc'>`Wallenrod</span>, <span class='clSrc'>Jero`zolima</span>, <span class='clSrc'>`Waszyngton</span>, <span class='clSrc'>A`meryka</span>;</li>
</ul>
<p>3) ударение может находиться на последнем слоге. Такое ударение имеют:</p>
<ul>
<li>слова иноязычного происхождения, как правило, заимствованные из французского языка: <span class='clSrc'>atelier</span> [ate`lie] «ателье», <span class='clSrc'>menu</span> [me`niu] «меню», <span class='clSrc'>renomé</span> [reno`me] «реноме», <span class='clSrc'>turneé</span> [tur`ne] «турне», <span class='clSrc'>interwiew</span> [inter`vju] «интервью», <span class='clSrc'>renault</span> [re`no] «рено»;</li>
<li>некоторые аббревиатуры: <span class='clSrc'>PZU</span> (<span class='clSrc'>Państwowy Zakład Ubezpieczeń</span> «Управление государственного страхования»), <span class='clSrc'>PKiN</span> (<span class='clSrc'>Pałac Kultury i Nauki</span> «Дворец культуры и науки»);</li>
<li>некоторые сложные слова, в состав которых входят корни <span class='clSrc'>arcy</span>, <span class='clSrc'>eks</span>, <span class='clSrc'>wice</span>: <span class='clSrc'>wice`król</span> «вицекороль», <span class='clSrc'>eks`mąż</span> «эксмуж», <span class='clSrc'>arсy`łotr</span> «архинегодяй».</li>
</ul>
<p>Существуют слова, имеющие две одинаково правильные ударные формы: <span class='clSrc'>`w ogóle / w o`góle</span> «вообще».</p>
<p>Рядом со словами, имеющими свое собственное ударение, существуют слова, собственного ударения не имеющие.</p>
<p>Безударные слова, присоединяющиеся к последующему слову с собственным ударением, называются проклитиками. Ими в польском языке могут быть:</p>
<ul>
<li>частица <span class='clSrc'>nie</span> в том случае, когда она связывается с многосложным словом: <span class='clSrc'>nie `czytam</span> «не читаю», <span class='clSrc'>nie `taki</span> «не такой», <span class='clSrc'>nie `Roman</span> «не Роман»;</li>
<li>предлоги перед многосложными словами: <span class='clSrc'>do `lasu</span> «в лес»;</li>
<li>односложные союзы: <span class='clSrc'>Powiedział, że `nie chce</span> «он сказал, что не хочет»; <span class='clSrc'>Przyjdź, jak `zechcesz</span> «приди, когда захочешь»;</li>
<li>местоимение co в сочетаниях типа: <span class='clSrc'>co `rok</span> «каждый год», <span class='clSrc'>co `dzień</span> «каждый день».</li>
</ul>
<p>Безударные слова, присоединяющиеся к предшествующему слову с собственным ударением, называются энклитиками. Ими в польском языке могут быть:</p>
<ul>
<li>односложные слова после частицы <span class='clSrc'>nie</span>, на которую подает ударение: <span class='clSrc'>`nie wiem</span> «не знаю», <span class='clSrc'>`nie chcę</span> «не хочу», <span class='clSrc'>`nie tak</span> «не так», <span class='clSrc'>`nie Jan</span> «не Ян»;</li>
<li>односложные формы существительных и местоимений после предлога: <span class='clSrc'>`do mnie</span> «ко мне», <span class='clSrc'>`na wsi</span> «в деревне»;</li>
<li>краткие формы личных местоимений: <span class='clSrc'>`mówię ci</span> «говорю тебе», <span class='clSrc'>`daj mi go</span> «дай мне его»;</li>
<li>частицы <span class='clSrc'>się, no, by</span>: <span class='clSrc'>`myje się</span> «моюсь», <span class='clSrc'>`daj no mi</span> «ну дай мне», <span class='clSrc'>`można by</span> «можно бы».</li>
</ul>
<p>Некоторые слова в одном контексте могут иметь самостоятельное ударение, в другом выступать в качестве клитик: <span class='clSrc'>`Ty tu zostań</span> «ты тут останься» - <span class='clSrc'>`Coś ty</span> «что ты»; <span class='clSrc'>On `nie chce</span> «он не хочет» - <span class='clSrc'>Nie `mogę</span> «не могу»; <span class='clSrc'>`Dasz mi coś</span> «дай мне что-нибудь» - <span class='clSrc'>`Nie dam</span> «не дам»; <span class='clSrc'>`Niech pan idzie</span> «идите» - <span class='clSrc'>Niech `kupi `cukru</span> «пусть он купит сахар».</p>
