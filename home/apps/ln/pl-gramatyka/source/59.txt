
=== Способы выражения направления действия

<h1>Способы выражения направления действия</h1>
<ul>
<li><span class='clSrc'>do</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>strzelać do celu</span> «стрелять в цель». Словосочетания этого типа соответствуют конструкциям <span class='clSrc'>w</span> + <span class='clSrc'>предл. пад.</span> со значением места, например: <span class='clSrc'>w szkole</span> - <span class='clSrc'>iść do szkoły</span>, <span class='clSrc'>w lesie</span> - <span class='clSrc'>iść do lasu</span>, <span class='clSrc'>w fabryce</span> - <span class='clSrc'>iść do fabryki</span>;</li>
<li><span class='clSrc'>род.пад.</span>; конструкции этого типа существуют параллельно сочетаниям с предлогом <span class='clSrc'>do</span>, например: <span class='clSrc'>dojść czegoś</span> / <span class='clSrc'>do czegoś</span> «дойти до чего-л.», <span class='clSrc'>dopłynąć czegoś /do czegoś</span> «доплыть до чего-л.», <span class='clSrc'>dobiec czego</span> «достичь чего-л.»;</li>
<li><span class='clSrc'>między</span> + <span class='clSrc'>вин.пад.</span> (+ <span class='clSrc'>a/i</span> + <span class='clSrc'>вин.пад.</span>), например: <span class='clSrc'>wleźć między krzesło a stół</span> «влезть между столом и стулом»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, соответствует конструкции <span class='clSrc'>na</span> + <span class='clSrc'>предл.пад.</span> для выражения места, например: <span class='clSrc'>na placu</span> - <span class='clSrc'>iść na plac</span>, <span class='clSrc'>na ulicy</span> - <span class='clSrc'>iść na ulicę</span>, <span class='clSrc'>na stadionie</span> - <span class='clSrc'>iść na stadion</span>, <span class='clSrc'>na Białorusi</span> - <span class='clSrc'>jechać na Białoruś</span>, <span class='clSrc'>na Litwie</span> - <span class='clSrc'>jechać na Litwę</span>, <span class='clSrc'>na Węgrzech</span> - <span class='clSrc'>jechać na Węgry</span>, <span class='clSrc'>na wsi</span> - <span class='clSrc'>jechać na wieś</span>, <span class='clSrc'>na lotnisku</span> - <span class='clSrc'>jechać na lotnisko</span>;</li>
<li><span class='clSrc'>(po)nad</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>wznieść się nad chmury</span> «подняться выше облаков», <span class='clSrc'>spojrzeć ponad siebie</span> «посмотреть вверх», <span class='clSrc'>przenieść się nad polską granicę</span> «перенестись к польской границе»;</li>
<li><span class='clSrc'>ku</span> + <span class='clSrc'>дат.пад.</span>, например: <span class='clSrc'>ku domowi</span> «к дому», <span class='clSrc'>ku ziemi</span> «к земле»;</li>
<li><span class='clSrc'>pod</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>podejść pod twierdzę</span> «подойти к крепости», <span class='clSrc'>wyjechać pod Kraków</span> «поехать в сторону Кракова», <span class='clSrc'>odprowadzić pod sam dom</span> «довести до самого дома»;</li>
<li><span class='clSrc'>przed</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>podjechać przed dom</span> «подъехать к дому», <span class='clSrc'>wyjść przed bramę</span> «выйти к воротам», <span class='clSrc'>wyrzucić przed siebie</span> «выбросить перед собой».</li>
</ul>
