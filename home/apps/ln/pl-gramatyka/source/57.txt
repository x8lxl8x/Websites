
=== Способы выражения времени действия

<h1>Способы выражения времени действия</h1>
<p>Значение времени в польском языке выражается многочисленными и разнообразными средствами, которые частично отличаются от соответствующих средств в русском языке. Эти отличия, в основном, касаются выбора падежной формы существительного и выбора предлога:</p>
<ul>
<li><span class='clSrc'>род.пад.</span> (при обязательном наличии сопровождающего определения), например: <span class='clSrc'>tego wieczora</span> «в этот вечер», <span class='clSrc'>tegoż dnia</span> «в тот же день», <span class='clSrc'>tej jesieni</span> «этой осенью», <span class='clSrc'>którejś niedzieli</span> «в какое-то воскресенье / однажды в воскресенье», <span class='clSrc'>tej samej chwili</span> «в этот момент», <span class='clSrc'>jednej nocy</span> «однажды ночью». Форма род.пад., как и в русском языке, употребляется для обозначения даты, например: <span class='clSrc'>Siostra przyjedzie dwunastego lutego</span> - «Сестра приедет двенадцатого февраля». Если действие отсутствует, и сообщение о дате является целью высказывания (например, в ответ на вопрос <span class='clSrc'>Którego jest dziś? / Którego mamy dzisiaj?</span>), параллельно употребляется конструкция с порядковым числительным в им.пад., например: <span class='clSrc'>Dziś jest dwunastego lutego</span> или <span class='clSrc'>Dziś jest dwunasty lutego</span> - «Сегодня двенадцатого февраля». В конструкциях с предлогами <span class='clSrc'>po, przed</span> порядковое числительное имеет падежную форму существительного, например: <span class='clSrc'>Ojciec wrócił przed / po dziewiątym sierpnia</span> - «Отец вернулся накануне / после девятого августа»;</li>
<li><span class='clSrc'>między</span> + <span class='clSrc'>твор.пад.</span> + <span class='clSrc'>a</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>między czwartą a piątą</span> «между четырьмя и пятью»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span> имеет несколько значений:</li>
<ol>
<li>соответствует русским предлогам <span class='clSrc'>на, до, к</span>, например: <span class='clSrc'>(umówić się) na (godzinę) siódmą</span> «(договориться) на шесть (часов)», <span class='clSrc'>(wykonać) na środę</span> «(выполнить) до среды / к среде»;</li>
<li>соответствует русскому предлогу <span class='clSrc'>в</span>, например: <span class='clSrc'>80 kilometrów na godzinę</span> «80 километров в час»;</li>
</ol>
<li><span class='clSrc'>na</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>na początku</span> «в начале», <span class='clSrc'>na końcu</span> «в конце»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span> + <span class='clSrc'>przed</span> + <span class='clSrc'>твор.пад.</span>, <span class='clSrc'>na</span> + <span class='clSrc'>(числительное)</span> + <span class='clSrc'>им.пад. / род.пад.</span> + <span class='clSrc'>przed</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>na mniej więcej tysiąc lat przed Chrystusem</span> «приблизительно за тысячу лет до рождения Христа», <span class='clSrc'>na miesiąc przed imieninami</span> «за месяц до именин», <span class='clSrc'>na rok przed Okrągłym Stołem</span> «за год до круглого стола», <span class='clSrc'>na trzy dni przed rozpoczęciem obrad</span> «за три дня до заседания»;</li>
<li><span class='clSrc'>nad</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>nad ranem</span> «к утру», <span class='clSrc'>nad wieczorem</span> «к вечеру»;</li>
<li><span class='clSrc'>raz</span> + <span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>raz na rok</span> «раз в год», <span class='clSrc'>raz na dwa tygodnie</span> «раз в два месяца»;</li>
<li><span class='clSrc'>o</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>o godzinie szesnastej</span> «в шестнадцать часов», <span class='clSrc'>o porze</span> «в пору», <span class='clSrc'>o świcie</span> «на рассвете»;</li>
<li><span class='clSrc'>o</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>spóżnić się o godzinę</span> «опоздать на час»;</li>
<li><span class='clSrc'>od</span> + <span class='clSrc'>род.пад.</span>, <span class='clSrc'>od</span> + <span class='clSrc'>наречие</span>, имеет два значения:</li>
<ol>
<li>«начиная с определенного момента времени», например: <span class='clSrc'>od soboty</span> «с субботы», <span class='clSrc'>od rana</span> «с утра», <span class='clSrc'>od dziecka</span> «с детства», <span class='clSrc'>od wczoraj</span> «со вчерашнего дня», <span class='clSrc'>od kiedy?</span> «с какого времени», <span class='clSrc'>od kiedy</span> «с того времени как», например: <span class='clSrc'>Od kiedy jestem w redakcji, nie mogę się nadziwić, ilu ludzi tu pracuje</span> - «С того времени как я работаю в редакции, я не перестаю удивляться, как много здесь работает людей»;</li>
<li>«на протяжении определенного отрезка времени», например: <span class='clSrc'>od trzech lat</span> «на протяжении трех лет», <span class='clSrc'>od dawna</span> «давно, издавна»; <span class='clSrc'>Nie palę już od roku</span> - «В течение года я уже не курю»;</li>
</ol>
<li><span class='clSrc'>od</span> + <span class='clSrc'>род.пад.</span> ... <span class='clSrc'>do</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>od (godziny) ósmej do czwartej</span> «с восьми до четырех (часов)»;</li>
</ul>
<blockquote>
<p>Русской конструкции <span class='clSrc'>от… до</span>, когда она используется для перечисления предметов (иногда она сопровождается употреблением обобщающих местоимений типа <span class='clSrc'>все</span>, <span class='clSrc'>всё</span>), соответствует польская конструкция <span class='clSrc'>od... po...</span> (очень редко - <span class='clSrc'>od ... do...</span>), например: «Он объездил всю Америку от Вашингтона до Филадельфии» - <span class='clSrc'>Objeździł całą Amerykę od Waszyngtonu po Filadelfię</span>; «Он знает всех польских философов - от Тарского до Пельца» - <span class='clSrc'>Zna wszystkich polskich filozofów od Tarskiego po Pelca</span>. Другая форма выражения того же самого значения - конструкция <span class='clSrc'>począwszy od</span> + <span class='clSrc'>род.пад.</span> … <span class='clSrc'>kończąc na</span> + <span class='clSrc'>предл.пад.</span></p>
</blockquote>
<ul>
<li><span class='clSrc'>po</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>po dziś dzień</span> «до сегодняшнего дня», <span class='clSrc'>po ostatnie tchnienie</span> «до последнего дыхания»;</li>
<li><span class='clSrc'>po</span> + <span class='clSrc'>предл.пад.</span>, употребляется только с глаголами прош.вр., например: <span class='clSrc'>Wróciłem po dwu latach</span> - «Я вернулся через три года»;</li>
<li><span class='clSrc'>po</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>po studiach</span> «после учебы», <span class='clSrc'>po rozmowie</span> «после разговора», <span class='clSrc'>po kawie</span> «после (того как выпили) кофе», <span class='clSrc'>po ranie</span> «после ранения»;</li>
<li><span class='clSrc'>przed</span> + <span class='clSrc'>твор.пад.</span>, имеет два значения: </li>
<ol>
<li>соответствует русскому сочетанию <span class='clSrc'>тому назад</span>, например: <span class='clSrc'>przed rokiem</span> «год тому назад», <span class='clSrc'>przed dwoma miesiącami</span> «два месяца тому назад», <span class='clSrc'>przed dwoma laty</span> «два года тому назад», <span class='clSrc'>przed laty</span> «много лет тому назад (давно)»;</li>
<li>употребляется, как русские предлоги <span class='clSrc'>до, перед, раньше</span>, например: <span class='clSrc'>(spotkać się) przed szóstą</span> «(встретиться) до шести часов», <span class='clSrc'>przed obiadem</span> «перед обедом / до обеда», <span class='clSrc'>przed wojną</span> «перед войной», <span class='clSrc'>przed czasem</span> «раньше времени»;</li>
</ol>
<li><span class='clSrc'>przez</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>przez cały wieczór</span> «целый вечер / на протяжении целого вечера», <span class='clSrc'>przez 25 lat</span> «на протяжении 25 лет»;</li>
<li><span class='clSrc'>przy</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>przy niedzieli</span> «в воскресенье», <span class='clSrc'>przy święcie</span> «в праздничный день», <span class='clSrc'>przy końcu lata</span> «в конце лета», <span class='clSrc'>przy kolacji</span> «во время ужина»;</li>
<li><span class='clSrc'>вин.пад.</span> + <span class='clSrc'>temu</span>, например: <span class='clSrc'>pięć lat temu</span> «пять лет тому назад», <span class='clSrc'>godzinę temu</span> «час тому назад»;</li>
<li><span class='clSrc'>w</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>w (owym) czasie</span> «в (то) время», <span class='clSrc'>w tej chwili</span> «в этот момент», <span class='clSrc'>w tym tygodniu</span> «на этой неделе», <span class='clSrc'>w nocy</span> «ночью». Ср. также: <span class='clSrc'>Myślałem o tym w drodze do Krakowa</span> - Я думал об этом по дороге в Краков;</li>
<li><span class='clSrc'>w/we</span> + <span class='clSrc'>вин.пад.</span>, имеет несколько значений: </li>
<ol>
<li>соответствует русскому предлогу <span class='clSrc'>в</span>, ср. сочетания с названиями дней недели: <span class='clSrc'>w środę</span> «в среду», <span class='clSrc'>we wtorek</span> «во вторник», структура которых сохраняется и при замене ед.ч. на мн.ч.: <span class='clSrc'>w środy</span> «по средам», <span class='clSrc'>we wtorki</span> «по вторникам». Сочетания с названиями месяцев не отличаются от соответствующих русских: <span class='clSrc'>w lutym</span>, <span class='clSrc'>w maju</span>, <span class='clSrc'>we wrześniu</span>. Существуют два вида конструкций со словом <span class='clSrc'>dzień</span>: <span class='clSrc'>w dzień</span> «днем», <span class='clSrc'>w dniu (urodzin)</span> «в день рождения»;</li>
<li>соответствует русскому предлогу <span class='clSrc'>за</span>, например: <span class='clSrc'>(przybiec) w godzinę</span> «прибежать за час», <span class='clSrc'>(skończyć) w dwa miesiące</span> «закончить за два месяца»;</li>
</ol>
<li><span class='clSrc'>w</span> + <span class='clSrc'>(числительное)</span> + <span class='clSrc'>вин.пад.</span> + <span class='clSrc'>наречие</span> или <span class='clSrc'>w</span> + <span class='clSrc'>(числительное)</span> + <span class='clSrc'>вин.пад.</span> + <span class='clSrc'>po</span> + <span class='clSrc'>предл.пад.</span> или <span class='clSrc'>w</span> + <span class='clSrc'>(числительное)</span> + <span class='clSrc'>вин.пад.</span> + <span class='clSrc'>od</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>w minutę później</span> «минутой / на минуту позже», <span class='clSrc'>w rok po Rosjanach</span> «на год позже, чем русские», <span class='clSrc'>w dwa miesiące od wizyty premiera</span> «через два месяца после визита премьера»;</li>
<li><span class='clSrc'>z</span> + <span class='clSrc'>род.пад.</span> + <span class='clSrc'>na</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>z pokolenia na pokolenie</span> «из поколения в поколение», <span class='clSrc'>z roku na rok</span> «из года в год».</li>
<li><span class='clSrc'>za</span> + <span class='clSrc'>вин.пад.</span>, употреблется только с глаголами в наст.вр. и прош.вр., например: <span class='clSrc'>za rok</span> «через год», <span class='clSrc'>za trzy godziny</span> «через три часа», <span class='clSrc'>za chwilę</span> «скоро», ср.: <span class='clSrc'>Za chwilę wrócę</span> - «Я скоро вернусь»;</li>
<li><span class='clSrc'>za</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>za caratu</span> «при царизме», <span class='clSrc'>za bolszewików</span> «при большевиках», <span class='clSrc'>za króla Stanisława Augusta Poniatowskiego</span> «при короле Станиславе Августе Понятовском».</li>
</ul>
<blockquote>
<p>Обозначения пор года:</p>
<ul>
<li>зимой - <span class='clSrc'>zimą, w zimie</span></li>
<li>весной - <span class='clSrc'>wiosną, na wiosnę</span></li>
<li>летом - <span class='clSrc'>latem, w lecie</span></li>
<li>осенью - <span class='clSrc'>jesienią, na jesieni</span></li>
</ul>
</blockquote>
