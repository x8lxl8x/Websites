
=== Способы выражения места действия

<h1>Способы выражения места действия</h1>
<ul>
<li><span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>włóczyć się wybrzeżem</span> «бродить по побережью», <span class='clSrc'>pójść górą</span> «пойти по горе», <span class='clSrc'>mknąć aleją</span> «мчаться по аллее», <span class='clSrc'>spływać asfaltem</span> «плыть по асфальту», <span class='clSrc'>przeciskać się korytarzem</span> «пробираться по коридору», <span class='clSrc'>biec schodami</span> «бежать по лестнице», <span class='clSrc'>posuwać się tropem</span> «идти по следу», <span class='clSrc'>wyskoczyć oknem (przez okno)</span> «выпрыгнуть через окно», <span class='clSrc'>przebiegało nią drżenie</span> «она дрожала всем телом / по ее телу пробежала дрожь»;</li>
<li><span class='clSrc'>obok</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>obok biurka</span> «возле стола»;</li>
<li><span class='clSrc'>na</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>położyć dłoń na ustach</span> «закрыть рот рукой»;</li>
<li><span class='clSrc'>nad</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>nad jeziorem</span> «около / у озера / на озере», <span class='clSrc'>nad morzem</span> «у моря / на море», <span class='clSrc'>nad Wisłą</span> «около / у Вислы, на Висле», <span class='clSrc'>nad Bałtykiem</span> «на Балтийском море»;</li>
<li><span class='clSrc'>po</span> + <span class='clSrc'>твор.пад.</span>, например: <span class='clSrc'>całować po rękach</span> «целовать руки», <span class='clSrc'>podrapać po głowie</span> «почесать голову», <span class='clSrc'>szukać po kieszeniach</span> «искать в карманах»;</li>
<li><span class='clSrc'>przy</span> + <span class='clSrc'>предл.пад.</span>, например: <span class='clSrc'>przy dworcu</span> «около вокзала», <span class='clSrc'>przy stole</span> «за столом»;</li>
<li><span class='clSrc'>u</span> + <span class='clSrc'>род.пад.</span>, например: <span class='clSrc'>szła u boku męża</span> «шла рядом с мужем», <span class='clSrc'>zawiesić dzwoneczek u szyi</span> «повесить колокольчик на шее»;</li>
<li><span class='clSrc'>w</span> + <span class='clSrc'>вин.пад.</span>, например: <span class='clSrc'>iść w jego ślady</span> «идти по его следам»;</li>
<li><span class='clSrc'>w</span> + <span class='clSrc'>предл.пад.</span>, <span class='clSrc'>na</span> + <span class='clSrc'>предл.пад</span>. Первый предлог обычно присоединяется к названию закрытого пространства (<span class='clSrc'>w szkole</span>, <span class='clSrc'>w domu</span>, <span class='clSrc'>w lesie</span>), а второй - к названию открытого пространства (<span class='clSrc'>na placu</span>, <span class='clSrc'>na ulicy</span>, <span class='clSrc'>na stadionie</span>, <span class='clSrc'>na lotnisku</span>). Это правило, однако, имеет свои весьма важные исключения. Традиционно считались нормативными конструкции: <span class='clSrc'>na Białorusi</span> «в Беларуси», <span class='clSrc'>na Litwie</span> «в Литве», <span class='clSrc'>na Kremlu</span> «в Кремле», <span class='clSrc'>na Krymie</span> «в Крыму», <span class='clSrc'>na Węgrzech</span> «в Венгрии», <span class='clSrc'>na Syberii</span> «в Сибири», <span class='clSrc'>na wsi</span> «в деревне», <span class='clSrc'>w fabryce</span> «на фабрике (в здании фабрики)». Но в последние годы в связи с тем, что некоторые страны Восточной Европы получили независимость (в частности, Литва, Украина, Беларусь), их новый государственный статус отразился в языке, а именно - в конструкциях со значением места. В современном польском языке получают все большее распространение сочетания <span class='clSrc'>w Litwie</span>, <span class='clSrc'>w Białorusi</span>, <span class='clSrc'>w Ukrainie</span> (по аналогии с выражениями <span class='clSrc'>w Polsce</span>, <span class='clSrc'>w Rosji</span>, <span class='clSrc'>w Argentynie</span>), хотя солидный и авторитетный «<em>Nowy słownik poprawnej polszczyzny</em>», изданный под редакцией проф. А. Марковского в 1999 г., не признавал эти новые формы как нормативные.</li>
</ul>
<blockquote>
<p>Глаголы размещения требуют в польском языке существительного в предл.пад., хотя в русском в аналогичной позиции находится существительное в вин.пад., ср.: <span class='clSrc'>położyć na stole</span> (<em>где?</em>) - «положить на стол» (<em>куда?</em>), <span class='clSrc'>powiesić w szafie</span> (где?) - «повесить в шкафу» (<em>куда?</em>), <span class='clSrc'>postawić na półce</span> (<em>где?</em>) - «поставить на полку» (<em>куда?</em>).</p>
</blockquote>
