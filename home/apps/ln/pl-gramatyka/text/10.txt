
=== Правописание gie - ge

<h1>Правописание gie - ge</h1>
<p>Сочетания букв <span class='clSrc'>gie</span> и <span class='clSrc'>ge</span> могут произноситься одинаково - как [g'e]. Их написание регулируется следующими правилами. Сочетание <span class='clSrc'>gie</span> пишем:</p>
<p><span class='clSrc'>а)</span> в словах польского происхождения, например: <span class='clSrc'>ogień</span> «огонь», <span class='clSrc'>drugie</span> «другое», <span class='clSrc'>długie</span> «долгое»;</p>
<p><span class='clSrc'>б)</span> в именах собственных литовского происхождения, например: <span class='clSrc'>Jagiełło</span>, <span class='clSrc'>Giedymin</span>, <span class='clSrc'>Olgierd</span>;</p>
<p><span class='clSrc'>в)</span> в существительных, которые заканчиваются на <span class='clSrc'>-giel, -gier, -gierz</span>, например: <span class='clSrc'>cyngiel</span> «курок», <span class='clSrc'>żagiel</span> «парус», <span class='clSrc'>ogier</span> «жеребец», <span class='clSrc'>szwagier</span> «шурин»;</p>
<p><span class='clSrc'>г)</span> в словах <span class='clSrc'>Algieria</span> «Алжир», <span class='clSrc'>Algierczyk</span> «алжирец», <span class='clSrc'>Algierka</span> «алжирка», <span class='clSrc'>algierka</span> «вид мужского пальто», <span class='clSrc'>algierski</span> «алжирский», <span class='clSrc'>Angielka</span> «англичанка», <span class='clSrc'>angielski</span> «английский», <span class='clSrc'>angielszczyzna</span> «английский язык», <span class='clSrc'>Archangielsk</span> «Архангельск», <span class='clSrc'>giełda</span> «биржа», <span class='clSrc'>megiera</span> «мегера», а также в словах, которые образованы от них.</p>
<p>Сочетание букв <span class='clSrc'>ge</span> (с мягким звучанием согласного) пишется в словах иноязычного происхождения <span class='clSrc'>germanista</span> [g'ermanista] «германист», <span class='clSrc'>generał</span> [g'eneraw] «генерал», <span class='clSrc'>geometria</span> [g'eometrja] «геометрия», <span class='clSrc'>algebra</span> [alg'ebra] «алгебра», <span class='clSrc'>geografia</span> [g'eografja] «география», <span class='clSrc'>georgina</span> [g'eorg'ina] «георгин», <span class='clSrc'>geranium</span> [g'eran'jum] «герань».</p>
<p>В некоторых заимствованных словах <span class='clSrc'>ge</span> в соответствии с нормой произносится как [ge], т.е. с твердым согласным, например: <span class='clSrc'>gehenna</span> [gexenna] «геена», <span class='clSrc'>getto</span> [getto] «гетто», <span class='clSrc'>gejsza</span> [gejša] «гейша», <span class='clSrc'>generacja</span> [generacja] «поколение», <span class='clSrc'>gest</span> [gest] «жест», <span class='clSrc'>gestapo</span> [gestapo] «гестапо», <span class='clSrc'>gestykulacja</span> [gestykulacja] «жестикуляция», <span class='clSrc'>tragedia</span> [tragedja] «трагедия», <span class='clSrc'>geneza</span> [geneza] «генезис», <span class='clSrc'>genealogia</span> [genealogja] «генеалогия».</p>
<p>В последнее время твердое произношение [g] перед гласным [e] под влиянием письменных текстов расширяется. Так, наряду с произношением [g'eneral] или [alg'ebra] распространяется произношение [general] и [algebra].</p>
