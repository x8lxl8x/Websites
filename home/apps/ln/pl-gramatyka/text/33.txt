
=== Словообразование прилагательных

<h1>Словообразование прилагательных</h1>
<p>В зависимости от того, от какой части речи образуются прилагательные, их, как и существительные, можно разделить на следующие группы:</p>
<ol>
<li>Прилагательные, образованные от существительных;</li>
<li>Прилагательные, образованные от прилагательных;</li>
<li>Прилагательные, образованные от глаголов;</li>
<li>Прилагательные, образованные от наречий;</li>
<li>Прилагательные, образованные от предложных сочетаний;</li>
<li>Сложные прилагательные.</li>
</ol>
<h2>Прилагательные, образованные от существительных</h2>
<p>Служат для определения существительного, с которым они согласуются, через отражение его связи с другим предметом (<span class='clSrc'>śnieżny człowiek</span> «снежный человек» - <span class='clSrc'>człowiek ze śniegu</span> «человек из снега»):</p>
<p><span class='clSrc'>а)</span> в наиболее общем виде это значение выражается при помощи следующих суффиксов:</p>
<ul>
<li><span class='clSrc'>-sk-, -ejsk-, -ijsk-, -yjsk-, -ańsk-, -ińsk-</span>: <span class='clSrc'>bankier</span> «банкир» - <span class='clSrc'>bankierski</span> «банкирский», <span class='clSrc'>Europa</span> - <span class='clSrc'>europejski</span> «европейский», <span class='clSrc'>Monachium</span> «Мюнхен» - <span class='clSrc'>monachijski</span> «мюнхенский», <span class='clSrc'>Kanada</span> - <span class='clSrc'>kanadyjski</span> «канадский», <span class='clSrc'>Afryka</span> - <span class='clSrc'>afrykański</span> «африканский». При присоединении этих суффиксов может наблюдаться стечение согласных с одновременным их изменением, например, <span class='clSrc'>biedniak-sk-i</span> - <span class='clSrc'>biedniacki</span> «бедняцкий», <span class='clSrc'>boż-sk-i</span> - <span class='clSrc'>boski</span> «божественный»;</li>
<li><span class='clSrc'>-ow-, -owsk-, -own-</span>: <span class='clSrc'>biuro</span> «бюро, офис» - <span class='clSrc'>biurowy</span> «офисный», <span class='clSrc'>czas</span> «время» - <span class='clSrc'>czasowy</span> «временный»; <span class='clSrc'>syn</span> «сын» - <span class='clSrc'>synowski</span> «принадлежащий сыну», <span class='clSrc'>Żyd</span> «еврей» - <span class='clSrc'>żydowski</span> «еврейский»; <span class='clSrc'>cud</span> «чудо» - <span class='clSrc'>cudowny</span> «чудесный», <span class='clSrc'>zysk</span> «прибыль» - <span class='clSrc'>zyskowny</span> «прибыльный»;</li>
<li><span class='clSrc'>-n-</span>: <span class='clSrc'>barwa</span> «цвет» - <span class='clSrc'>barwny</span> «разноцветный», <span class='clSrc'>bieda</span> «беда, нищета» - <span class='clSrc'>biedny</span> «бедный», <span class='clSrc'>bok</span> «сторона» - <span class='clSrc'>boczny</span> «боковой, побочный»;</li>
<li><span class='clSrc'>-cz-</span>: <span class='clSrc'>opiekun</span> «опекун» - <span class='clSrc'>opiekuńczy</span> «заботливый»;</li>
</ul>
<p><span class='clSrc'>б)</span> следующие суффиксы вносят значение наличия того, что обозначается словом, от которого произведено прилагательное, иногда в качестве основной, перевешивающей остальные, характеристики:</p>
<ul>
<li><span class='clSrc'>-at-, -owat-</span>: <span class='clSrc'>żona</span> «жена» - <span class='clSrc'>żonaty</span> «женатый», <span class='clSrc'>dziecie</span> «дети» - <span class='clSrc'>dzieciaty</span> «имеющий детей», <span class='clSrc'>broda</span> «борода» - <span class='clSrc'>brodaty</span> «бородатый», <span class='clSrc'>węzeł</span> «узел» - <span class='clSrc'>węzłowaty</span> «узловатый»;</li>
<li><span class='clSrc'>-ast-, -ist-, -yst-</span>: <span class='clSrc'>głowa</span> «голова» - <span class='clSrc'>głowiasty</span> «головастый», <span class='clSrc'>igła</span> «игла» - <span class='clSrc'>iglasty</span> «иглистый» (как видно, этот суффикс смягчает основу), <span class='clSrc'>kula</span> «шар» - <span class='clSrc'>kulisty</span> «шаровидный», <span class='clSrc'>barki</span> «плечи» - <span class='clSrc'>barczysty</span> «плечистый»;</li>
<li><span class='clSrc'>-aw-, -iw-, -yw-, -liw-</span>: <span class='clSrc'>kędziorу</span> «кудри» - <span class='clSrc'>kędzierzawy</span> «кудрявый», <span class='clSrc'>parsza</span> «парша» - <span class='clSrc'>parszywy</span> «паршивый», <span class='clSrc'>choroba</span> «болезнь» - <span class='clSrc'>chorobliwy</span> «болезненный», <span class='clSrc'>wstyd</span> «стыд» - <span class='clSrc'>wstydliwy</span> «стыдливый»;</li>
</ul>
<p><span class='clSrc'>в)</span> на материал, из которого сделан тот или иной предмет, указывают прилагательные, образованные с помощью следующих суффиксов:</p>
<ul>
<li><span class='clSrc'>-an-</span>: <span class='clSrc'>drewno</span> «бревно, лесоматериал» - <span class='clSrc'>drewiany</span> «деревянный», <span class='clSrc'>szkło</span> «стекло» - <span class='clSrc'>szklany</span> «стеклянный»;</li>
<li><span class='clSrc'>-ow-</span>: <span class='clSrc'>porcelana</span> «фарфор» - <span class='clSrc'>porcelanowy</span> «фарфоровый», <span class='clSrc'>guma</span> «резина» - <span class='clSrc'>gumowy</span> «резиновый».</li>
</ul>
<h2>Прилагательные, образованные от прилагательных</h2>
<p>Делятся на две группы, противоположные по значению:</p>
<p><span class='clSrc'>а)</span> первые обозначают ослабление признака:</p>
<ul>
<li><span class='clSrc'>-aw-</span>: <span class='clSrc'>biały</span> «белый» - <span class='clSrc'>białawy</span> «беловатый», <span class='clSrc'>tępy</span> «тупой» - <span class='clSrc'>tępawy</span> «туповатый», у одного из прилагательных суффикс представлен в слегка измененном виде: <span class='clSrc'>zielony</span> - <span class='clSrc'>zielonkawy</span> «зеленоватый»;</li>
<li><span class='clSrc'>-owat-</span>: <span class='clSrc'>ciemny</span> «темный» - <span class='clSrc'>ciemnowaty</span> «темноватый», <span class='clSrc'>głupi</span> «глупый» - <span class='clSrc'>głupowaty</span> «глуповатый»;</li>
</ul>
<p><span class='clSrc'>б)</span> другие обозначают усиление признака:</p>
<ul>
<li><span class='clSrc'>-utki-</span>: <span class='clSrc'>mały</span> «малый» - <span class='clSrc'>malutki</span> «маленький», <span class='clSrc'>chudy</span> «худой» - <span class='clSrc'>chudziutki</span> «худенький»;</li>
<li><span class='clSrc'>-eńk-</span>: <span class='clSrc'>mały</span> «малый» - <span class='clSrc'>maleńki</span> «маленький», <span class='clSrc'>cały</span> «целый» - <span class='clSrc'>caleńki</span> «целенький». Как видно, суффиксы этой группы используются для того, чтобы придать слову ласкательную окраску и обычно присоединяются к словам <span class='clSrc'>miły</span> и <span class='clSrc'>mały</span>: <em>mal<span class='clSrc'>uchn</span>y</em>, <em>mal<span class='clSrc'>un</span>i</em>, <em>mal<span class='clSrc'>uczk</span>i</em>, <em>mal<span class='clSrc'>us</span>i</em>, <em>mal<span class='clSrc'>uśk</span>i</em>, <em>mal<span class='clSrc'>usieńk</span>i</em>, <em>mal<span class='clSrc'>usieczk</span>i</em>, <em>mal<span class='clSrc'>usienieczk</span>i</em>, <em>mal<span class='clSrc'>uteńk</span>i</em>, <em>mal<span class='clSrc'>utenieczk</span>i</em>, <em>mac<span class='clSrc'>iupeńk</span>i</em>, <em>mac<span class='clSrc'>iupc</span>i</em>, <em>mac<span class='clSrc'>iupk</span>i</em>.</li>
</ul>
<p>Как и для существительного, приставочный способ образования не является типичным для прилагательного, приставка присоединяются к основам, имеющим в своем составе суффикс - <span class='clSrc'>bez-dziet-n-y</span> «бездетный», <span class='clSrc'>bez-akcent-ow-y</span> «безударный». Ниже даны некоторые основные приставки, в большинстве случаев их значения совпадают со значением соответствующих русских:</p>
<ul>
<li><span class='clSrc'>bez-</span>: <span class='clSrc'>bezbożny</span> «безбожный», <span class='clSrc'>bezbarwny</span> «бесцветный»;</li>
<li><span class='clSrc'>do-</span>: <span class='clSrc'>dożylny</span> «внутривенный», <span class='clSrc'>domęśniowy</span> «внутримышечный» (приставка, соответствующая русской <span class='clSrc'>внутри-</span>);</li>
<li><span class='clSrc'>między-</span>: <span class='clSrc'>międzymiastowy</span> «междугородный», <span class='clSrc'>międzyżebrowy</span> «межреберный»;</li>
<li><span class='clSrc'>nie-</span>: <span class='clSrc'>niewinny</span> «невинный», <span class='clSrc'>niesmaczny</span> «невкусный»;</li>
<li><span class='clSrc'>po-</span>: <span class='clSrc'>pograniczny</span> «пограничный», <span class='clSrc'>pośmiertny</span> «посмертный» (ср. также примеры, в которых приставка соответствует русской приставке <span class='clSrc'>после-</span>: <span class='clSrc'>powojenny</span> «послевоенный», <span class='clSrc'>pooperacyjny</span> «послеоперационный»);</li>
<li><span class='clSrc'>pod-</span>: <span class='clSrc'>podniebny</span> «поднёбный», <span class='clSrc'>podskórny</span> «подкожный»;</li>
<li><span class='clSrc'>poza-</span>: <span class='clSrc'>pozabiurowy</span> «внеслужебный», эта приставка также соответствует русской <span class='clSrc'>сверх-</span>: <span class='clSrc'>pozalimitowy</span> «сверхлимитный» (ср. также еще одно соответствие приставки <span class='clSrc'>сверх-</span>: <span class='clSrc'>ponaddzwiękowy</span> «сверхзвуковой»);</li>
<li><span class='clSrc'>przed-</span>: <span class='clSrc'>przedhistoryczny</span> «доисторический», <span class='clSrc'>przedchrześcijański</span> «дохристианский»;</li>
<li><span class='clSrc'>przeciw-</span>: <span class='clSrc'>przeciwczołgowy</span> «противотанковый», <span class='clSrc'>przeciwtyfusowy</span> «противотифозный»;</li>
<li><span class='clSrc'>przy-</span>: <span class='clSrc'>przydrożny</span> «придорожный», <span class='clSrc'>przydworcowy</span> «привокзальный».</li>
</ul>
<h2>Прилагательные, образованные от глаголов</h2>
<p>Имеют разнородные значения: с одной стороны, они приписывают существительному, с которым согласуются, способность выполнять действие, названное производящим глаголом (<span class='clSrc'>odpowiedzieć</span> «ответить» - <span class='clSrc'>odpowiedzialny człowiek</span> «ответственный человек»), с другой стороны, показывают обращенность действия на существительное (<span class='clSrc'>jadać</span> «есть (многократно)» - <span class='clSrc'>jadalny pokój</span> «комната для еды (столовая)»):</p>
<ul>
<li><span class='clSrc'>-n-, -aln-</span>: <span class='clSrc'>wędrować</span> «странствовать, путешествовать» - <span class='clSrc'>wędrowny</span> «странствующий», <span class='clSrc'>zależeć</span> «зависеть» - <span class='clSrc'>zależny</span> «зависимый», <span class='clSrc'>mieszkać</span> «жить, проживать» - <span class='clSrc'>mieszkalny</span> «жилой», <span class='clSrc'>błagać</span> «молить, просить» - <span class='clSrc'>blagalny</span> «умоляющий»;</li>
<li><span class='clSrc'>-it-, -yt-, -ist-, -yst-</span>: <span class='clSrc'>należeć</span> «принадлежать» - <span class='clSrc'>należyty</span> «надлежащий», <span class='clSrc'>wyrazić</span> «выразить» - <span class='clSrc'>wyrazisty</span> «выразительный», <span class='clSrc'>sprężyć</span> «спружинить» - <span class='clSrc'>sprężysty</span> «пружинистый»;</li>
<li><span class='clSrc'>-iw-, -liw-</span>: <span class='clSrc'>chcieć</span> «хотеть» - <span class='clSrc'>chciwy</span> «жадный, алчный», <span class='clSrc'>godzić się</span> «соглашаться» - <span class='clSrc'>godziwy</span> «дозволенный, допустимый», <span class='clSrc'>bóść</span> «бодать» - <span class='clSrc'>bodliwy</span> «бодливый», <span class='clSrc'>dokuczać</span> «докучать» - <span class='clSrc'>dokuczliwy</span> «докучливый»;</li>
<li><span class='clSrc'>-k-</span>: <span class='clSrc'>sypać</span> «сыпать» - <span class='clSrc'>sypki</span> «сыпкий», <span class='clSrc'>grząść</span> «вязнуть» - <span class='clSrc'>grząski</span> «вязкий».</li>
</ul>
<h2>Прилагательные, образованные от наречий</h2>

<li><span class='clSrc'>-n-</span>: <span class='clSrc'>zewnątrz</span> «снаружи» - <span class='clSrc'>zewnętrzny</span> «наружный»;</li>

<h2>Прилагательные, образованные от предложных сочетаний</h2>
<p>Основой для образования прилагательного может быть существительное в каком-либо падеже с предлогом: <span class='clSrc'>mimo woli</span> - <span class='clSrc'>mimowolny</span> «мимовольный», <span class='clSrc'>przed potopem</span> - <span class='clSrc'>przedpotopowy</span> «допотопный», <span class='clSrc'>bez pana</span> - <span class='clSrc'>bezpański</span> «бесхозный», <span class='clSrc'>bez kłopotu</span> - <span class='clSrc'>bezkłopotliwy</span> «безхлопотный».</p>
<h2>Сложные прилагательные</h2>
<p>Как и существительные, прилагательные могут образовываться путем сложения основ, которые иногда принадлежат к разным частям речи: <span class='clSrc'>prawda</span>, <span class='clSrc'>mówić</span> «говорить» - <span class='clSrc'>prawdomówny</span> «правдивый», <span class='clSrc'>biały</span>, <span class='clSrc'>pióro</span> «перо» - <span class='clSrc'>białopióry</span> «белоперый», <span class='clSrc'>głuchy</span>, <span class='clSrc'>niemy</span> - <span class='clSrc'>głuchoniemy</span> «глухонемой».</p>
