
=== Правописание j - i

<h1>Правописание j - i</h1>
<p>Звук [j] может обозначаться буквами <span class='clSrc'>j</span> и <span class='clSrc'>i</span>. Буква <span class='clSrc'>j</span> пишется:</p>
<p><span class='clSrc'>а)</span> в начале слова, например: <span class='clSrc'>jezioro</span> «озеро», <span class='clSrc'>jesień</span> «осень», <span class='clSrc'>jutro</span> «завтра», <span class='clSrc'>język</span> «язык», <span class='clSrc'>jeść</span> «есть»;</p>
<p><span class='clSrc'>б)</span> после гласных, например: <span class='clSrc'>nadzieja</span> «надежда», <span class='clSrc'>zając</span> «заяц», <span class='clSrc'>moja</span> «моя», <span class='clSrc'>pije</span> «пьет»;</p>
<p><span class='clSrc'>в)</span> после согласных <span class='clSrc'>c, z, s</span>, например: <span class='clSrc'>stacja</span> «станция», <span class='clSrc'>operacja</span> «операция», <span class='clSrc'>Rosja</span> «Россия», <span class='clSrc'>poezja</span> «поэзия», <span class='clSrc'>dygresja</span> «экскурс»;</p>
<p><span class='clSrc'>г)</span> после приставок <span class='clSrc'>nad-, ob-, od-, pod-, przed-, w-, zd-</span>, например: <span class='clSrc'>nadjechać</span> «подъехать», <span class='clSrc'>objąć</span> «обнять», <span class='clSrc'>objętość</span> «объем», <span class='clSrc'>odjechać</span> «отъехать», <span class='clSrc'>wjazd</span> «въезд», <span class='clSrc'>zdjąć</span> «снять», <span class='clSrc'>zdjęcie</span> «снимок».</p>
<p>Буква <span class='clSrc'>i</span> для обозначения звука [j] пишется перед гласными после согласных (кроме <span class='clSrc'>c, z, s</span>), например:</p>

<li><span class='clSrc'>miniatura</span> - миниатюра</li>
<li><span class='clSrc'>linia</span> - линия</li>

<li><span class='clSrc'>radio</span> - радио </li>
