====== Имя числительное

=== Количественные числительные

<h6>Имя числительное</h6>

<h1>Количественные числительные</h1>

<p><span class='clSrc'>jeden</span> «один» - изменяется по родам (<span class='clSrc'>jeden</span>, <span class='clSrc'>jedna</span>, <span class='clSrc'>jedno</span>) и по падежам по образцу прилагательного. Выступая как часть сложного числительного, не склоняется (<span class='clSrc'>do dwudziestu jeden lat</span> «до двадцати одного года»).</p>
<p>Эта лексема может иметь формы множественного числа (<span class='clSrc'>jedni</span>, <span class='clSrc'>jedne</span>). В этом случае она перестает быть числительным и начинает выполнять функцию неопределенного местоимения (<span class='clSrc'>jedni państwo</span> - <span class='clSrc'>pewni państwo</span>). Значение неопределенности свойственно всем формам числительного <span class='clSrc'>jeden</span>, но для данных оно является единственно возможным.</p>

<p>Числительные <span class='clSrc'>jedenaście</span> - <span class='clSrc'>dziewiętnaście</span>, <span class='clSrc'>dwadzieścia</span> - <span class='clSrc'>czterdzieści</span>, <span class='clSrc'>dwieście</span> перед окончаниями косвенных падежей меняют мягкий согласный на твердый (<span class='clSrc'>jedenaście</span> - <em>jedenastu, jedenastoma</em>).</p>
<p>Числительные <span class='clSrc'>pięćdziesiąt</span> - <span class='clSrc'>dziewięćdziesiąt</span>, наоборот, перед окончаниями косвенных падежей меняют твердый согласный на мягкий (<span class='clSrc'>dziewięćdziesiąt</span> - <em>dziewięćdziesięciu</em>).</p>
<p>У числительных <span class='clSrc'>dwanaście</span>, <span class='clSrc'>dwadzieścia</span>, <span class='clSrc'>dwieście</span> склоняются обе части (<span class='clSrc'>dwanaście</span> - <em>dwunastu</em>);</p>
<p>Наоборот, у числительных <span class='clSrc'>pięćdziesiąt</span> - <span class='clSrc'>dziewięćdziesiąt</span> склоняется только последняя часть (<span class='clSrc'>pięćdziesiąt</span> - <em>pięćdziesięciu</em>); а у форм <span class='clSrc'>pięćset</span> - <span class='clSrc'>dziewięćset</span> склоняется только первая часть (<span class='clSrc'>pięćset</span> - <em>pięciuset</em>).</p>
<p>Составные числительные образуются путем простого соположения чисел: 12345 - <span class='clSrc'>dwanaście tysięcy trzysta czterdzieści pięć</span>. У составных числительных склоняются все компоненты (<span class='clSrc'>dwudziestu pięciu, siedmiu tysięcy siedmiuset siedemdziesięciu siedmiu</span>).</p>
<blockquote>
<p>Исключение - числительное <span class='clSrc'>jeden</span>, которое в составе составных числительных не склоняется и не изменяется по родам (<span class='clSrc'>trzydzieści jeden kobiet</span> «тридцать одна женщина», <span class='clSrc'>do pięćdziesięciu jeden lat</span> «до пятидесяти одного года»).</p>
</blockquote>
