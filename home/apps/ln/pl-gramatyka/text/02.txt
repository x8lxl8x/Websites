
=== Диграфы

<h1>Диграфы</h1>
<p>Кроме отдельных букв для передачи некоторых звуков употребляются диграфы - комбинации из двух букв. В польской графике насчитывается семь диграфов.</p>
<table>
<tr>
<th>Диграф</th>
<th>Транскрипция</th>
<th>Примеры</th>
</tr>
<tr>
<td><span class='clSrc'>cz</span></td>
<td>[č]</td>
<td><span class='clSrc'>czysty</span> «чистый», <span class='clSrc'>człowiek</span> «человек», <span class='clSrc'>lecz</span> «но»</td>
</tr>
<tr>
<td><span class='clSrc'>dz</span></td>
<td>[dz]</td>
<td><span class='clSrc'>władza</span> «власть», <span class='clSrc'>widzę</span> «вижу», <span class='clSrc'>sadza</span> «сажа»</td>
</tr>
<tr>
<td><span class='clSrc'>dź</span></td>
<td>[dž']</td>
<td><span class='clSrc'>niedźwiedź</span> «медведь», <span class='clSrc'>dźwięk</span> «звук», <span class='clSrc'>chodźmy</span> «давай пойдем»</td>
</tr>
<tr>
<td><span class='clSrc'>dż</span></td>
<td>[dž]</td>
<td><span class='clSrc'>jeżdżę</span> «езжу», <span class='clSrc'>dżdżysty</span> «дождливый», <span class='clSrc'>dżuma</span> «чума»</td>
</tr>
<tr>
<td><span class='clSrc'>ch</span></td>
<td>[x]</td>
<td><span class='clSrc'>chory</span> «больной», <span class='clSrc'>pech</span> «неудача», <span class='clSrc'>chłopski</span> «крестьянский»</td>
</tr>
<tr>
<td><span class='clSrc'>rz</span></td>
<td>[ž], [š]</td>
<td><span class='clSrc'>morze</span> «море», <span class='clSrc'>krzesło</span> «стул», <span class='clSrc'>drzewo</span> «дерево»</td>
</tr>
<tr>
<td><span class='clSrc'>sz</span></td>
<td>[š]</td>
<td><span class='clSrc'>szósty</span> «шестой», <span class='clSrc'>Warszawa</span> «Варшава», <span class='clSrc'>czaszka</span> «череп»</td>
</tr>
</table>
<p>Каждый из польских диграфов <span class='clSrc'><em>dz</em></span>, <span class='clSrc'><em>dź</em></span>, <span class='clSrc'><em>dż</em></span> обозначает смычно-щелевой звук того же типа, что и русский звук [ц]. Впрочем, и в русской речи встречается звук [дз], например, в фамилии <em>Шевардна<span class='clSrc'>дз</span>е</em>, в сочетании <em>оте<span class='clSrc'>ц</span> дома</em> (мы произносим [атедздома]). Но в некоторых случаях диграфы <span class='clSrc'>dz</span>, <span class='clSrc'>dż</span> обозначают не один, а два звука, например:</p>
<ul>
<li><span class='clSrc'>nadzwyczajny</span> [nad-zwyčajny] - чрезвычайный</li>
<li><span class='clSrc'>podzamcze</span> [pod-zamče] - посад</li>
<li><span class='clSrc'>odzyskać</span> [od-zyskač'] - найти, отыскать</li>
<li><span class='clSrc'>odznaka</span> [od-znaka] - награда, отличие</li>
<li><span class='clSrc'>odzwierciedlić</span> [od-zv'erč'edl'ič'] - отразить</li>
<li><span class='clSrc'>podżartować</span> [pod-žartovač'] - пошутить</li>
<li><span class='clSrc'>podżegać</span> [pod-žegač'] - подталкивать</li>
<li><span class='clSrc'>podżywić</span> [pod-žyv'ič'] - подкармливать</li>
</ul>
<p>Диграф <span class='clSrc'>rz</span> чаще всего обозначает согласный звук [ž], например, в словах <span class='clSrc'>rzadko</span> «редко», <span class='clSrc'>rzecz</span> «вещь, дело», <span class='clSrc'>porządek</span> «порядок», <span class='clSrc'>marzec</span> «март», <span class='clSrc'>rzucać</span> «бросать».</p>
<blockquote>
<p>После глухих согласных <span class='clSrc'>rz</span> всегда произносится как глухой звук [š], например: <span class='clSrc'>przestrzeń</span> [pšestšen'] «пространство», <span class='clSrc'>trzeba</span> [tšeba] «надо», <span class='clSrc'>powietrze</span> [pov'etše] «воздух», <span class='clSrc'>krzyk</span> [kšyk] «крик», <span class='clSrc'>chrzan</span> [xšan] «хрен». Это правило сохраняется и для буквы <span class='clSrc'>w</span>: после глухих согласных она читается как [f], например: <span class='clSrc'>twój</span> [tfuj] «твой», <span class='clSrc'>twardy</span> [tfardy] «твердый», <span class='clSrc'>czwarty</span> [čfarty] «четвертый».</p>
<p>Но в некоторых случаях <span class='clSrc'>rz</span> надо произносить иначе - как [rz]: <span class='clSrc'>marznąć</span> «мерзнуть», <span class='clSrc'>zmarzły</span> «замерзший», <span class='clSrc'>erzac</span> «эрзац», <span class='clSrc'>zmierznąć</span> «опротиветь», <span class='clSrc'>zmierzły</span> «противный».</p>
</blockquote>
<p>Диграф <span class='clSrc'>ch</span> в сочетании с буквой <span class='clSrc'>s</span> в некоторых заимствованных словах произносится как [sx'], например: <span class='clSrc'>schizofrenia</span> «шизофрения», <span class='clSrc'>schizofrenik</span> «шизофреник», <span class='clSrc'>ischias</span> (вариант написания: <span class='clSrc'>iszjas</span>) «ишиас» (произносится также как [išjas]), <span class='clSrc'>schizma</span> (устаревший вариант написания: <span class='clSrc'>szyzma</span>) «схизма, раскол» (произносится также как [šyzma]).</p>
